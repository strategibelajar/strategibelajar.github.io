Jawaban:

4. n(S) = 180 orang

n(M) = 103 orang

n(B) = 142 orang

n(MB)= x orang

180 = 103 + 142 - x

x = 245 - 180

x = 65

Jadi, yang lulus adalah 65 orang

5. Banyak siswa yg gemar keduanya = a

Total Siswa = Gemar IPA + Gemar IPS + Tidak Gemar IPA & IPS - Gemar IPA & IPS

36 = 20 + 15 + 8 - a

36 = 43 - a

a = 43 - 36

a = 7 siswa

Jadi, banyak siswa yang gemar keduanya ada 7 siswa.

c. f(-2) = -4

f(x) = ax + b

f(3) = 3a + b = 1

f(1). = a + b = -1

---------------- -

2a = 2

a = 2/2 = 1

a + b = -1

1 + b = -1

b = -1 - 1 = -2

a. a = 1

b = -2

b. f(x) = x - 2

c. f(-2) = -2 - 2 = -4

Penjelasan dengan langkah-langkah:

f(x) = ax + b

f(3) = 1 ➡ 3a + b = 1

f(1) = -1 ➡ a + b = -1

_______________–

2a = 2

a = 1

Substitusikan nilai ke dalam salah satu persamaan

a + b = -1

1 + b = -1

b = -1 - 1

b = -2

a. Nilai a dan b adalah 1 dan -2

b. Bentuk fungsi ➡ f(x) = x - 2

c. Nilai dari f(-2)

f(-2) = -2 - 2
