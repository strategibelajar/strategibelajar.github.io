6. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

16. Pemerintah menyediakan 7.915 kursi Untuk calon Ppk Guru. Formasi Guru yang diterima ada 2.550 guru SD, guru matematika sebanyak 2.015, guru bahasa Inggris sebanyak 885 guru, dan guru bahasa Indonesia sebanyak 1.020. Sisanya untuk guru ips dan guru IPA, Guru IPA 34 lebih banyak daripada Guru ips. Banyak Guru ips Adalah.​..


Jumlah guru IPS adalah 705,5 orang. Jumlah sisa guru untuk guru ips dan guru IPA adalah 1445 orang

Penjelasan dengan langkah-langkah

Diketahui:

Total formasi  7.915 kursi

Formasi guru SD = 2550 kursi

Formasi guru b.inggris = 885 kursi

Formasi guru matematika = 2015 kursi

Formasi guru b. indonesia = 1020 kursi.

Guru Ipa lebih banyak 34 orang

Ditanyakan:

Berapakah formasi untuk guru IPS

Jawaban

Menghitung sisa kursi untuk guru IPA dan IPS

2.550 (Guru SD) + 2.015 (Guru Matematika) + 885 (Guru Bahasa Inggris) + 1.020 (Guru Bahasa Indonesia) = 6.470 guru

Sisa kursi = total formasi - jumlah formasi yang telah dialokasikan =  7.915  - 6.470 = 1445

Menghitung jumlah guru IPS

Guru IPS + Guru IPA =  1445

x + x + 34 =  1445

2 x =  1445 - 34

x = 705,5

[tex]\frac{1}{30}[/tex] x [tex]\frac{1}{2}[/tex] = [tex]\frac{1 . 1}{30 . 2}[/tex]

[tex]\frac{1}{30}[/tex] x [tex]\frac{1}{2}[/tex] = [tex]\frac{1}{60}[/tex]
