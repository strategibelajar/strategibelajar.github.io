c. f(-2) = -4

Penjelasan dengan langkah-langkah:

f(x) = ax + b

f(3) = 3a + b = 1

f(1). = a + b = -1

---------------- -

2a = 2

a = 2/2 = 1

a + b = -1

1 + b = -1

b = -1 - 1 = -2

a. a = 1

b = -2

b. f(x) = x - 2

c. f(-2) = -2 - 2 = -4

Penjelasan dengan langkah-langkah:

f(x) = ax + b

f(3) = 1 ➡ 3a + b = 1

f(1) = -1 ➡ a + b = -1

_______________–

2a = 2

a = 1

Substitusikan nilai ke dalam salah satu persamaan

a + b = -1

1 + b = -1

b = -1 - 1

b = -2

a. Nilai a dan b adalah 1 dan -2

b. Bentuk fungsi ➡ f(x) = x - 2

c. Nilai dari f(-2)

f(-2) = -2 - 2

f(-2) = -4
