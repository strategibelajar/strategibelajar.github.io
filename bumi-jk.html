20. Hasil pengukuran kapasitas panas C suatu zat padat sebagai fungsi temperatur T dinyatakan oleh persamaan C = aT + at + ß³. Satuan untuk a dan ß yang mungkin adalah.A. J untuk a dan JK-2 untuk B B. JK² untuk a dan J untuk B C. JK untuk a dan JK³ untuk B D. JK-2 untuk a dan JK-4 untuk BBantu.​


Jawaban:

D. JK-2 untuk a dan JK-4 untuk B

Penjelasan:

Satuan C yaitu J/K, jika C = aT, maka a = C/T sehingga satuannya a = J/K² atau JKˉ², dan jika C = ß³T maka ß = C/T³, sehingga satuan ß = J/K⁴ atau JKˉ⁴. Jadi,Jawabannya adalah D.
