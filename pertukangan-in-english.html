x = tukang kayu

y = tukang batu

---------------------------_

x = -150.000 / -2

x = 75.000

3x + y = 275.000

3 (75.000) + y = 275.000

225.000 + y = 275.000

y = 275.000 - 225.000

y = 50.000

2x + 3y = 2(75.000) + 3(50.000)

= 150.000 + 150.000

= 300.000

Jadi, upah 2 tukang kayu dan 3 tukang batu adalah Rp 300.000

25. terdapat 90 tukang ojek dan tukang becak di kompleks Bumi Asri. Jika banyak tukang ojek adalah 54, maka perbandingan tukang ojek dan tukang becak adalah?? sma cara,a ya... :)


tukang ojek+tukang becak = 90
tukang ojek = 54
maka perbandingan antara tukang ojek : tukang becak
tukang ojek : tukang becak
54 : (90-54)
54 : 36
FPB 54 dan 36 adalah 18
maka sederhanakan (54:18) : (36:18) = 3 : 2 Dik : Terdapat 90 tukang ojek dan becak di komplek Bumi Asri serta banyak tukang ojek adalah 54.
Dit : Berapa perbandingan tukang ojek dan tukang becak?
Jawab:
Tukang Ojek = 54 orang
Tukang Becak = 90 - 54 = 36 orang
Maka, perbandingan tukang ojek : tukang becak = 54 : 36 = 3 : 2

Jadi, perbandingan tukang ojek dan tukang becak adalah 3 : 2

tukang sapu = y
Berlaku persamaan
4x + 2y = 220000 .... Pers (1)
3x + y = 140000 ........Pers (2)

4x + 2y = 220000
6x + 2y = 280000
Kurangkan (Pers 2 - Pers 1), sehingga
2x = 60000
x = 30000
Substitusikan x=30000 ke Pers 1
4x + 2y = 220000
4. 30000 + 2y = 220000
120000 + 2y = 220000
2y= 220000-120000= 100000
y= 100000/2=50000

2x + y = 2. 30000 + 50000 = 110000
Jadi insentif untuk 2 tukang kebun dan seorang tukang sapu adalah Rp 110000, 00
