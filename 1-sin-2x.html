Pembahasan

Rumus Selisih Kuadrat

a² - b² = (a - b)(a + b)

Identitas Trigonometri :

cos²x + sin²x = 1

cos²x = 1 - sin²x

sin²x = 1 - cos²x

Akan dibuktikan bahwa

1 - sin⁴x = cos²x (1 + sin²x)

1 - sin⁴x

= (1 - sin²x)(1 + sin²x)

= ( cos²x + sin²x - sin²x)(1 + sin²x)

= cos²x (1 + sin²x)

cos²x (1 + sin²x)

= (1 - sin²x)(1 + sin²x)

= 1 - sin⁴x

1 - sin⁴x = cos²x (1 + sin²x)

23. tentukan Hp persamaan sin x + sin 2x =0 dengan menggunakan:1. relasi sudut di kuadrat 1 dan kuadrat 3 - sin 2x = sin (180+2x 2. relasikan sudut di kuadrat 1 dengan kuadrat 4 - sin2x = sin (360-2x)​


Jawaban:

sin 2x = sin 120° untuk 0° ≤ x ≤ 180° adalah {30°, 60°}. Nilai tersebut diperoleh dari perhitungan dengan rumus penyelesaian persamaan trigonmetri. Simak pembahasan berikut.

Pembahasan

Diketahui:

sin 2x = sin 120° untuk 0° ≤ x ≤ 180°

Ditanya: Himpunan penyelesaian untuk x

Jawab:

Penyelesaian persamaan trigonometri untuk sinus

sin x = sin α

Maka

x = α + k × 360°

x = (180° - α) + k × 360°

dengan k = 0, 1, 2, 3, 4, ...

Dari soal diketahui sin 2x = sin 120° untuk 0° ≤ x ≤ 180°, maka diperoleh

sin 2x = sin 120° untuk 0° ≤ x ≤ 180°

cara 1:

sin 2x = sin 120°

2x = 120° + k × 360°

x = 120°/2 + k × 360°/2

x = 60° + k × 180°

untuk k = 0

x = 60° + k × 180°

x = 60° + 0 × 180°

x = 60° + 0°

x = 60°

untuk k = 1

x = 60° + k × 180°

x = 60° + 1 × 180°

x = 60° + 180°

x = 240° (tidak memenuhi)

cara 2:

sin 2x = sin 120°

2x = (180° - 120°) + k × 360°

2x = 60° + k × 360°

x = 60°/2 + k × 360°/2

x = 30° + k × 180°

untuk k = 0

x = 30° + k × 180°

x = 30° + 0 × 180°

x = 30° + 0°

x = 30°

untuk k = 1

x = 30° + k × 180°

x = 30° + 1 × 180°

x = 30° + 180°

x = 210° (tidak memenuhi)

Jadi himpunan penyelesaian dari x adalah {30°, 60°}

Trigonometri adalah sebuah metode matematika yang membandingkan antara sisi dan sudut. Trigonometri digunakan dalam matematika untuk mencari panjang sisi lain dengan membandingkan sisi dengan sudut yang telah diketahui.

Trigonometri ini sendiri terbagi menjadi 6, diantaranya sinus (sin), cosinus (cos), tangen (tan), cosecan (csc), secotan (sec), cotangen (cot).
