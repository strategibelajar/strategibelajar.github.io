2. berapa integral cot²x integral (sin x - cos x)(sin x + cos x) tolong bantu


Integral of Trigonometry Functions.

∫ cot² x dx
= ∫ (csc² x - 1) dx
= - cot x - x + C
= -1 / tan x - x + C

= ∫ (sin² x - cos² x) dx
= ∫ (-cos 2x) dx
= -1/2 sin 2x + C
= -cos x sin x + C

3. Integral cos√x adalah


Penting:
Refresh apabila jawaban masih berupa kode.

Dengan menggunakan integral substitusi umum:
Maka:
[tex]$\begin{align}u&=\sqrt{x}\xrightarrow{\text{Diferensiasikan}}&du&=\frac{1}{2\sqrt{x}}\, dx \\ &&du&=\frac{1}{2u}\, dx \\ &&dx&=2u\, du\end{align}[/tex]

Sehingga, berlaku:
[tex]$\begin{align}\int\cos\sqrt{x}\, dx&=\int \cos u(2u\, du) \\ &=\int 2u\cos u\, du\end{align}[/tex]

Lakukan integrasi parsial dengan:
[tex]$\begin{align}\int \underbrace{2u}_{U}\underbrace{\cos u\, du}_{dV}&=\underbrace{2u}_{U}\underbrace{\sin u}_{V}-\int\underbrace{\sin u}_{V}\underbrace{2\, du}_{dU} \\ &=2u.\sin u-\int 2\sin u\, du \\ &=2u.\sin u-2(-\cos u)+C \\ &=2u.\sin u+2\cos u+C \\ &=2\sqrt{x}.\sin\sqrt{x}+2\cos\sqrt{x}+C&&\text{|| Substitusikan}\end{align}[/tex]

10. tentukan hasil pengintegralan. integral x^2 cos x dx


Integral Parsial.

∫ u dv = uv - ∫ v du
u = x² → du = 2x dx
dv = cos x → v = sin x
∫ x² cos x dx = x² sin x - ∫ 2x sin x dx
                    = x² sin x - 2 ∫ x sin x dx

∫ f dg = fg - ∫ g df
f = x → df = dx
dg = sin x → g = -cos x
∫ x sin x dx = x sin x + ∫ cos x dx
                 = -x cos x + sin x

∫ x² cos x dx = x² sin x - 2(sin x - x cos x) + C
                     = (x² - 2) sin x + 2x cos x + C
