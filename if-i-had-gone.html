[tex]Answer[/tex]

1.  I wouldn't have missed the bus if we  had gone earlier.

2.  If I were not a teacher, I couldn't teach  you.

3. If Mr.Frank had had time, he would have attended the seminar in Malaysia.

[tex]Explanation[/tex]

Conditional sentence structure consists -If clause and result clause ( main clause ).

→There are 4 types of Conditional Sentence←

ZERO CONDITIONAL

The Zero Conditional is used to describe the Phenomenon, habit, and General truth ( Fact  ). Zero Conditional uses the simple present tense form to reflect the truth.

Pattern :

If+S+verb (s/es)+O , S+verb (s/es)+O

Example :

If we freeze the water in the freezer , it changes into an ice.

{ Jika kita mendinginkan air di freezer, itu berubah menjadi es }

Conclusion:

→ It uses the simple present tense form to describe the phenomenon. It is proven that the water changes into an ice when it is put in the freezer.

TYPE 1 CONDITIONAL

The Type 1 Conditional is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened ). In fact, the speaker will actualize his/her dream now or soon .

Pattern :

If +S+ verb (s/es) + O, S+will+verb1+O

Example :

She will be mad if I take her note.

{ Dia akan marah jika aku ambil catatanya}

Conclusion :

→ It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true. Meaning/Fact : it's possible that she will be mad.

TYPE 2 CONDITIONAL

The Type 2 Conditional type  is used to describe  the untruth in the present / future. It is contrary to the fact in the present or it could be in the future. The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future

"Keep in mind, if we use the first , second person and third person singular  ( I, she, he, it , ) in the nominal sentence , it must be followed by the PAST linking verb "were."" It is the same way to the plural form as the formal writing." If we use the PAST linking verb "was " , it should be used in the colloquial as the informal conversation.

Pattern:

If+S+verb2+O/C , S+would/could+verb1+O/C

Example :

If I were not a teacher, I couldn't teach you.

{ Jika Aku bukan seorang guru, aku tidak bisa mengajarimu }

Conclusion :

→ It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future. Meaning/Fact :  I am a teacher, so I can teach you well.

TYPE 3 CONDITIONAL

The Type 3 Conditional  is used to describe  the untruth in the past. It is contrary to the fact in the past and It  reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb 3+O  , S+Would/could+have+Verb 3+O

Example :

I wouldn't have missed the bus if we  had gone earlier.

{ Aku tidak akan telah ketinggalan bis jika kita berangkat lebih awal } 

Conclusion :

It reflects the regret for something that couldn't be reached in the past. Meaning/Fact :  I missed the bus because we didn't go earlier.

If Mr.Frank had had time, he would have attended the seminar in Malaysia

Conclusion :

Meaning/Fact : Mr. Frank didn't have time, so he didn't attend the seminar in Malaysia.

15. If I had not gone to bed so late,i would not have been tired yang ditanya adalah faktanya... berarti???


Conditional Sentence

Fact: I was tired because I went to bed very late.

＿＿＿＿＿＿＿＿

Preface

Conditional sentence atau kalimat pengandaian adalah suatu kalimat majemuk yang menyatakan bahwa peristiwa dalam main clause (induk kalimat; klausa tanpa if) hanya akan terjadi jika syarat yang dinyatakan dalam if clause (klausa yang diawali if) terpenuhi.

Contoh:

Saya akan datang jika dia mengundang saya.

I will come if she invites me.

Ada 3 tipe kalimat pengandaian, yaitu:

Type 1 → mungkin akan terjadi di masa depan (future possible)Type 2 → kemungkinan tidak akan terjadi karena syaratnya tidak sedang dipenuhi sekarang (present unreal)Type 3 → tidak mungkin terjadi karena merujuk pada kejadian di masa lalu (past unreal)

Bandingkan ketiga kalimat berikut:

Type 1 → We will go to the beach tomorrow if it doesn't rain.Type 2 → We would go to the beach today if it didn't rain.Type 3 → We would have gone to the beach yesterday if it hadn't rained.

＿＿＿＿＿＿＿＿

Explication

If I had not gone to bed so late, I would not have been tired.

Jika saya tidak pergi tidur sangat larut, saya tidak akan kelelahan.

Kalimat di atas merupakan conditional sentence type 3 karena menggunakan past perfect pada main clause dan if clausenya menggunakan would have + past participle.

Karena type 3 adalah kalimat pengandaian tentang kejadian yang seharusnya namun tidak terjadi di masa lalu, maka fakta harus berlawanan dengan kalimat pengandaian dan menggunakan past tense.

Jadi, faktanya adalah:

I was tired because I went to bed very late.

Saya kelelahan karena saya pergi tidur sangat larut.

＿＿＿＿＿＿＿＿
