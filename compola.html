8. ada yang bisa bantu materi tentang Passe Compose​


LE PASSÉ COMPOSÉ

1. Je suis rentré à 11 heures.

2. Il a fait la cuisine?

3. Nous avons fini la cuisine.

4. L’avion a décollé à midi.

5. Anne et Sophie sont venues en voiture.

------------------

Le passé composé adalah salah satu temps dalam bahasa Perancis.

Le passé composé digunakan untuk menyatakan suatu keadaan atau aksi yang telah selesai di masa lalu dan menekankan pada hasil keadaan/aksi tersebut di masa sekarang.

Struktur kalimat dalam passé composé adalah:

sujet + avoir/être + participe passé

Kata kerja bantu avoir/être menggunakan bentuk présent.

Kata kerja bantu être hanya digunakan bersama 14 kata kerjadi bawah ini:

naître/mourir (lahir/meniggal)aller/venir (pergi/datang)monter/descendre (menaiki/menuruni)arriver/partir (tiba/berangkat)entrer/sortir (masuk/keluar)apparaître (muncul)rester (tinggal)retourner (kembali)tomber (jatuh)

dan turunannya seperti

revenir (pulang)rentrer (masuk kembali)remonterredescendrerepartir

24. Hasil compile dari JDK dalam mengcompile bitycode dari Java menjadi


Penjelasan:

Menulis kode program Java. Untuk program Java sederhana, Anda bisa menggunakan editor sederhana seperti Notepad, Notepad++ atau yang lain. Untuk program Java yang sangat kompleks, Anda dapat menggunakan IDE seperti Eclipse, Intellij, NetBean atau JDeveloper.

Mengkompilasi (compile) file kode sumber Java (Java source code) yang berekstensi .java. Untuk program Java sederhana dapat menggunakan Command Prompt Windows namun harus sudah dikonfigurasi dengan benar. Untuk mengkompilasi program Java yang kompleks dapat menggunakan salah satu IDE yang sudah disebutkan sebelumnya.

Kemudian menjalankan (run) file bytecode Java yang berekstensi .class. Tentu saja, menjalankan file bytecode Java di Command Prompt dan di IDE memiliki cara atau langkan yang berbeda.

Berikut ini adalah contoh program Java dengan nama file Sapaan yang akan menampilkan teks atau string "Selamat Datang Java" di jendela Command Prompt (dulu Dos Prompt). Kode program Java ini ditulis menggunakan text editor Notepad++ dan selanjutnya akan dikompilasi dan dijalankan di jendela Command Prompt.
