Lakukan browsing di internet menggunakan mesin pencari google, kemudian buatlah daftar kendala apa saja yang kalian alami selama melakukan pencarian data di internet Kesimpulan Kendala / Masalah saat mengakses internet 1 Server not found 2 Browser loading terus menerus 3 Tampilan layar tidak utuh 4 Page time out 5 Error 404 Kesimpulannya adalah: bahwa kendala ketika kita berselancar di internet sangat beragam tergantung dari kondisi jaringan internet dan juga perangkat yang dipakai. Penilaian meliputi: 1. Membuat daftar inventaris permasalahan berselancar di internet 2. Membuat kesimpulan dari permasalah data yang di inventaris atas permasalahan berselancar di internet Skor Penilaian: ● SB = Sangat Baik ● B = Baik ● C = Cukup ● K = Kurang


Jawaban:

Inventaris Permasalahan saat Berselancar di Internet:

1. Server not found: Terjadi ketika server yang dituju tidak dapat ditemukan, sehingga halaman tidak dapat diakses.

2. Browser loading terus menerus: Ketika halaman web membutuhkan waktu yang lama untuk dimuat atau tidak dapat dimuat dengan sempurna.

3. Tampilan layar tidak utuh: Halaman web tidak tampil dengan baik atau elemen-elemen tertentu tidak terlihat dengan jelas.

4. Page time out: Proses pemuatan halaman memakan waktu terlalu lama sehingga browser memberikan pesan time out.

5. Error 404: Halaman yang dicari tidak ditemukan atau tidak tersedia.

Kesimpulan:

Ketika menggunakan internet, kita dapat menghadapi berbagai kendala seperti server tidak ditemukan, masalah pemuatan halaman, tampilan yang tidak sempurna, halaman tidak ditemukan, dan lain sebagainya. Kendala ini dapat disebabkan oleh masalah jaringan internet, performa perangkat yang digunakan, atau masalah pada situs web yang diakses. Oleh karena itu, penting untuk memahami dan mengatasi kendala-kendala ini agar dapat menjelajahi internet dengan lancar.

Skor Penilaian:

B (Baik)
