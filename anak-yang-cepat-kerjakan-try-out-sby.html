BC² = AB² + AC² - 2AB.AC.cos x

4. Empat orang anak yang bernama Rudi, Gilang, Yoga, dan Bima mengikuti Try Out Mata Pelajaran Matematika di sekolahnya. Jika rata-rata nilai Try Out Rudi dan Gilang adalah 50, rata-rata nilai Try Out Gilang dan Yoga adalah 75, serta rata-rata nilai Try Out Yoga dan Bima adalah 70. Berapakah rata-rata nilai Try Out Rudi dan Bima?​


Untuk menemukan rata-rata nilai Try Out Rudi dan Bima, kita perlu menggunakan informasi yang diberikan.

Kita diberi tiga persamaan:

(Rudi + Gilang) / 2 = 50

(Gilang + Yoga) / 2 = 75

(Yoga + Bima) / 2 = 70

Kita selesaikan persamaan ini untuk mencari nilai rata-rata Try Out Rudi dan Bima.

Dari persamaan 1

(Rudi + Gilang) / 2 = 50, kita bisa mendapatkan Rudi + Gilang = 100.

Dari persamaan 2

(Gilang + Yoga) / 2 = 75, kita bisa mendapatkan Gilang + Yoga = 150.

Dari persamaan 3

(Yoga + Bima) / 2 = 70, kita bisa mendapatkan Yoga + Bima = 140.

Kita ingin mencari rata-rata Rudi dan Bima, yaitu (Rudi + Bima) / 2.

Jika kita mengurangi persamaan Gilang + Yoga = 150 dari persamaan Rudi + Gilang = 100, maka kita mendapatkan (Rudi - Yoga) = -50.

Jika kita mengurangi persamaan Yoga + Bima = 140 dari persamaan Gilang + Yoga = 150, maka kita mendapatkan (Gilang - Bima) = 10.

Jadi, kita memiliki sistem persamaan:

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50Gilang - Bima = 10

Kita ingin mencari nilai (Rudi + Bima) / 2. Untuk mencapainya, kita dapat menambah persamaan di atas:

(Rudi - Yoga) + (Gilang - Bima) = (-50) + 10

Rudi + Bima = -40

Akhirnya, kita dapat mencari rata-rata nilai Try Out Rudi dan Bima:

(Rudi + Bima) / 2 = (-40) / 2 = -20

Jadi, rata-rata nilai Try Out Rudi dan Bima adalah -20.

Xn + 95 = 88n + 88

Xn = 88n - 7...(2)

eliminasi deh.

Xn = 88n - 7

0 = -3n + 12

-3n = -12

n = 12/3

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.

Penjelasan:

Diketahui:

AC = 7 cm

ABC siku-siku di C

CD merupakan garis tinggi

Ditanya:

Panjang CD

Jika tambahan diketahui: (1) BC = 5 dan (2) BD = 3

Jawab:

Untuk pernyataan (1) BC = 5

Panjang AB = [tex]\sqrt{AC^2+BC^2}[/tex]

                    = [tex]\sqrt{7^2+5^2}[/tex]

                    = [tex]\sqrt{49+25}[/tex]

                    = [tex]\sqrt{74}[/tex]

Panjang CD = [tex]\frac{AC*BC}{AB}[/tex]

                    = [tex]\frac{7*5}{\sqrt{74} }[/tex]

Untuk pernyataan (2) BD = 3

AD² + CD² = AC²    ⇒  AD² + CD² = 49

BD² + CD² = BC²    ⇒     9   + CD² = BC²

                                    ______________ -

                                    AD² - 9 = 49 - BC²

                                         BC² = 49 + 9 - AD²

                                         BC² = 58 - AD²

Pada segitiga ABC berlaku

A C ²  +  B C ²  =  A B ²

7 ²  + 58 - AD²   = ( A D  + 3)²

107 - AD²   = AD² + 6AD + 9

2AD² + 6AD - 98 = 0

Faktorkan, maka akan ketemu panjang AD

Jika Panjang AD sudah diketahui, Maka panjang CD juga dapat kita cari dengan pythagoras.

Sehingga jawabannya:

Pilihan yang sesuai adalah  ( D )  Pernyataan  ( 1 )  SAJA cukup untuk menjawab pertanyaan, dan pernyataan  ( 2 )  SAJA cukup.
