1. Tentukan Un dari : 2 , 5 , 11, 20, 32, 47, ...​


Jawaban:

72

masing-masing angka ditambah 3

2....5......11.....20......32......47....

..+3...+6....+9....+12....+15

......+3....+3....+3.....+3

2a = 3

a = 3/2

3a + b = 3

3(3/2) + b = 3

9/2 + b = 6/2

b = 6/2 - 9/2

b = -3/2

a + b + c = 2

3/2 + (-3/2) + c = 2

3/2 - 3/2 + c = 2

0 + c = 2

c = 2

Un = an^2 + bn + c

Un = 3/2 n^2 + (-3/2)n + 2

Un = (3/2)n^2 - (3/2)n + 2

Un = 1/2(3n^2 - 3n + 4)

jadi Un nya adalah Un = 1/2(3n^2 - 3n + 4)

38. 32. Nilai dari 3-4(3)¹ 3² +5(2)° + 47° +2-2 2-2(2)² adalah ....​


Penjelasan dengan langkah-langkah:

Untuk menyelesaikan ungkapan matematika ini, mari kita uraikan setiap bagian dan hitung nilainya secara berurutan:

1. Mulai dengan 3 - 4(3)¹. Dalam hal ini, kita akan atur pangkat 3¹ menjadi 1, maka kita dapat menulisnya ulang menjadi: 3 - 4(3) = 3 - 4(3) = 3 - 12 = -9.

2. Selanjutnya, kita punya 3². Pangkat di sini adalah ², yang berarti kita harus mengalikan 3 dengan dirinya sendiri. Jadi, 3² = 3 × 3 = 9.

3. Kemudian, kita memiliki 5(2)°. Pangkat di sini adalah °, yang berarti kita mengangkat 2 ke pangkat 0. Pangkat 0 adalah 1, sehingga 5(2)° = 5 × 1 = 5.

4. Berikutnya, kita punya 47°. Karena pangkatnya adalah °, itu berarti kita mengangkat 47 ke pangkat 0, yang juga menghasilkan 1. Jadi, 47° = 1.

5. Setelah itu, kita punya 2-2. Dalam hal ini, kita harus atur dua ekspresi -2 menjadi 1, sehingga 2-2 = 2^(-2) = 1/2^2 = 1/4.

6. Terakhir, kita memiliki 2-2(2)². Kita mulai dengan menghitung pangkat tertinggi 2², yang sama dengan 4. Kemudian, kita mencari nilai dari -2(4), yaitu -8. Jadi, 2-2(2)² = 2 - 8 = -6.

Sekarang, kita dapat menggabungkan semua hasil perhitungan kita:

-9 + 9 + 5 + 1 + (1/4) - 6 = -9 + 9 + 5 + 1/4 - 6 = 0.25.

Jadi, nilai dari ungkapan matematika 3-4(3)¹ 3² +5(2)° + 47° +2-2 2-2(2)² adalah 0.25.
