10. Jika sin²3°+sin²6°+sin²9°+....+sin²87°+sin²90°=x Berapakah nilai x???


sin² 3° + sin² 6° + sin² 9° +.... + sin² 87° + sin² 90° = x

Sekarang pasangkan

(sin² 3° + sin² 87°) + (sin² 6° + sin² 84°) + (sin² 9° + sin² 81°) + .... (sin² 38° + sin² 52°) + (sin² 41° + sin² 49°) + (sin² 44° + sin² 46°) + sin² 45° +  sin² 90° = x

Ingat sifat ini :
sin a = cos (90 - a), maka :

(sin² 3° + cos² 3°) + (sin² 6° + cos² 6°) + (sin² 9° + cos² 9°) + .... (sin² 38° + cos² 38°) + (sin² 41° + cos² 41°) + (sin² 44° + cos² 44°) + sin² 45° +  sin² 90° = x

Ingat identitas ini :
sin² x° + cos² x° = 1, maka :

1 + 1 + 1 + ...... (Sebanyak 15 kali) + (½√2)² + 1² = x
15 + ½ + 1 = x
16,5 = x

Jadi, nilai x adalah 16.5

Trigonometri
Memeriksa apakah 3 sin x tan x = 3 sec x – sin x​ benar.

3 sin x tan x
= 3 sin x (sin x / cos x)
= (3 sin² x) (1 / cos x)
= 3 sin² x sec x
= 3 sec x sin² x
= 3 sec x (1 – cos² x)
= 3 sec x – 3(sec x cos² x)
= 3 sec x – 3(cos² x / cos x)
= 3 sec x – 3 cos x
