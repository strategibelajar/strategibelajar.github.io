1. y - n = 5y + n = 41y?n?​


y = 23

n = 18

PENYELESAIAN

Diketahui:

y – n = 5 y + n = 41

Ditanya:

nilai y? nilai x?

Jawab:

Mencari y

y – n = 5

y + n = 41

_________ +

2y = 46

y = 46 ÷ 2

y = 23

Mencari n

Substitusikan y = 23 ke salah satu persamaan di atas

y – n = 5

23 – n = 5

–n = 5 – 23

–n = –18

n = 18

PEMBUKTIAN

Untuk membuktikan apakah nilai n = 18 dan y = 23 benar atau tidak adalah dengan mensubsitusikan kedua nilai tersebut ke dalam persamaan

y – n = 5

23 – 18 = 5 ✔

y + n = 41

23 + 18 = 41 ✔

Matematika

______________________________Jawaban :

[tex]n = 18 \\ y = 23[/tex]

Penjelasan dengan langkah - langkah penyelesaian :

Perbandingan = Suatu cara yang digunakan untuk membandingkan dua bilangan atau lebih nilai ataupun itu objek, menggunakan cara yang paling sederhana.

Fungsi perbandingan = Mencari suatu bentuk atau nilai yang paling sederhana dari suatu nilai atau objek, dengan syarat bentuk pembanding dan salah satu nilai diketahui.

∆ Simbol perbandingan membandingkan nilai atau objek = Tanda titik dua ( : )

∆ Simbol perbandingan membandingkan pecahan = Lebih kecil ( < ), lebih besar ( > ) , & sama dengan ( = )

y - n = 5

y + n = 41

y ? n ?

Pembahasan :

Diketahui :

y - n = 5y + n = 41

Ditanya :

Nilai y = ... ?Nilai n = ... ?

Jawab :

[tex]y - n = 5 \\ y + n = 41[/tex]

[tex]y = 5 + n \\ n = 41 - n[/tex]

[tex]5 + n = 41 - n[/tex]

[tex]n + n = 41 - 5[/tex]

[tex]2n = 36[/tex]

[tex]n = 18[/tex]

[tex]y = 41 - 18[/tex]

[tex]y = 23[/tex]

[tex]n = 18 \\ y = 23[/tex]

Jadi, nilai n nya adalah 18 & nilai y adalah 23

∆ Pembuktian

[tex]y - n = 5 \\ y + n = 41[/tex]

[tex]23 - 18 = 5 \\ 23 + 18 = 41[/tex]

2. Jika X dan y adalah dua himpunan berhingga, sehingga n(X U Y) = 36, n(X) = 20, n(Y) = 28, maka tentukan n( X ∩ Y).Jelaskan.Diketahui: n(X) = 20, n (Y) = 28, n (X U Y) = 36.Karena itu, n(X U Y) = n(X) + n(Y) – n(X ∩ Y)


Nilai dari n (X ∩ Y) adalah 12 (dua belas). Arti dari n (X ∩ Y) adalah jumlah anggota himpunan X dan Y yang sama.

Penjelasan konsep dengan contoh, sebagai berikut:

Anggota dari himpunan X dan Y:
X = {2, 3, 4}
Y = {3, 4, 5, 6, 7}
Maka:
n (X) = 3 dan n (Y) = 5.Adapun X ∩ Y = {3, 4}.
Hal ini karena 2 dan 3 sama-sama ada di X dan Y.Maka:
n (X ∩ Y) = jumlah anggota himpunan X irisan Y.
n (X ∩ Y) = 2.
Hanya ada dua anggota saja, yaitu 3 dan 4.

Penjelasan dengan langkah-langkah

Tanda "∪" memiliki makna union atau gabungan. Maka keterangan himpunan X ∪ Y merujuk pada gabungan keseluruhan anggota himpunan X dan anggota himpunan Y. Sedangkan tanda "∩" memiliki makna intersection atau irisan. Maka keterangan himpunan X ∩ Y merujuk pada anggota himpunan X yang sama dengan anggota himpunan Y.

Diketahui:

n (X) = 20.n (Y) = 28.n (X ∪ Y) = 36.

Ditanyakan:

n (X ∩ Y) = ?

Penyelesaian:

n (X ∩ Y) = n(X) + n(Y) – n(X ∩ Y)

n (X ∩ Y) = 20 + 28 - 36

n (X ∩ Y) = 48 - 36

n (X ∩ Y) = 12.

Kesimpulan

Nilai dari n (X ∩ Y) adalah 12.

10. Tentukan semua bioangan bulat positif x,y,z dan n yang memenuhi persamaan:x^n+y^n=z^n


Jawaban:

Persamaan ini dikenal sebagai persamaan Fermat, yang tidak memiliki solusi bilangan bulat positif jika n>2. Oleh karena itu, jika n=1 atau n=2, kita dapat mencari semua solusi bilangan bulat positif (x, y, z) yang memenuhi persamaan tersebut.

Jika n=1, persamaannya menjadi x+y=z. Kita dapat memilih nilai x=1 dan y=2, sehingga z=3, atau memilih x=2 dan y=1, sehingga z=3. Solusi lainnya adalah permutasi dari solusi tersebut, yaitu x=1 dan y=3, sehingga z=4, atau x=3 dan y=1, sehingga z=4.

Jika n=2, persamaannya menjadi x^2+y^2=z^2. Ini adalah persamaan Pythagoras, yang memiliki banyak solusi bilangan bulat positif, seperti (3, 4, 5), (5, 12, 13), dan (8, 15, 17).
