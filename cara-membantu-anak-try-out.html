4. Tolong bantu. Lg try out


Mapel : Fisika
Bab : Gaya

EF = F3 - F2 - F1
EF = 1.500 - 600 - 600 N
EF = 1.500 - 1.200 N
EF = 300 N

W = 18.000 Joule

w = F × s
s = w/F

s = 18.000/300
s = 180/3
s = 60 meter

(D )Mapel : IPA Fisika
Kelas : 8 SMP/VIII SMP
Bab : Gaya dan Usaha 

Pembahasan Soal :
Jika usaha bersama gaya gaya tersebut terhadap benda 18.000 J, Benda berpindah sejauh..

Pembahasan Jawaban :
DiKetahui :
F1 = 600 N ( Kearah Kiri )
F2 = 600 N ( Kearah Kiri )
F3 = 1.500 N ( Kearah Kanan )
w = 18.000 J 

DiTanya :
s =...?

DiJawab :
Sebelum mencari s. Kita harus mencari ∑F
Rumus mencari ∑F
Yaitu : 
∑F = F3 - ( F1 + F2 )
∑F = 1.500 - ( 600 + 600 )
∑F = 1.500 - 1.200
∑F = 300 N ( Kearah Kanan )

Rumus mencari s 
Yaitu :
w = ∑F × s
18.000 = 300 × s
s = 18.000 : 300
s = 60 Meter.
Jadi, benda berpindah sejauh 60 meter ( D )

Semoga Membantu

Mapel : Fisika
Bab: Bidang Miring

W × h = F × s
10 × 0,5 = F × 2
10 × 5/10 = F × 2
5 = F × 2
F = 5/2
F = 2,5 N

(D)Mapel : IPA Fisika
Kelas : 8 SMP/VIII SMP
Bab : Pesawat Sederhana ( Bidang Miring )

Pembahasan Soal :
Sebuah Troli yang beratnya 10 N ditarik dari dasar bidang miring hingga mencapai puncak bidang miring. Besar gaya F Yang di perlukan adalah...

Pembahasan Jawaban :
DiKetahui :
W = 10 N 
h = 0,5 meter
s = 2 meter

DiTanya :
F =...?

DiJawab :
Rumus mencari F 
Yaitu :
W × h = s × F
F = W × h : s
F = 10 × 0,5 : 2
F = 5 : 2 
F = 2,5 N
Jadi,besar gaya yang dibutuhkan adalah 2,5 Newton ( D )

35. Empat orang anak yang bernama Rudi, Gilang, Yoga, dan Bima mengikuti Try Out Mata Pelajaran Matematika di sekolahnya. Jika rata-rata nilai Try Out Rudi dan Gilang adalah 50, rata-rata nilai Try Out Gilang dan Yoga adalah 75, serta rata-rata nilai Try Out Yoga dan Bima adalah 70. Berapakah rata-rata nilai Try Out Rudi dan Bima?​


Untuk menemukan rata-rata nilai Try Out Rudi dan Bima, kita perlu menggunakan informasi yang diberikan.

Kita diberi tiga persamaan:

(Rudi + Gilang) / 2 = 50

(Gilang + Yoga) / 2 = 75

(Yoga + Bima) / 2 = 70

Kita selesaikan persamaan ini untuk mencari nilai rata-rata Try Out Rudi dan Bima.

Dari persamaan 1

(Rudi + Gilang) / 2 = 50, kita bisa mendapatkan Rudi + Gilang = 100.

Dari persamaan 2

(Gilang + Yoga) / 2 = 75, kita bisa mendapatkan Gilang + Yoga = 150.

Dari persamaan 3

(Yoga + Bima) / 2 = 70, kita bisa mendapatkan Yoga + Bima = 140.

Kita ingin mencari rata-rata Rudi dan Bima, yaitu (Rudi + Bima) / 2.

Jika kita mengurangi persamaan Gilang + Yoga = 150 dari persamaan Rudi + Gilang = 100, maka kita mendapatkan (Rudi - Yoga) = -50.

Jika kita mengurangi persamaan Yoga + Bima = 140 dari persamaan Gilang + Yoga = 150, maka kita mendapatkan (Gilang - Bima) = 10.

Jadi, kita memiliki sistem persamaan:

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50Gilang - Bima = 10

Kita ingin mencari nilai (Rudi + Bima) / 2. Untuk mencapainya, kita dapat menambah persamaan di atas:

(Rudi - Yoga) + (Gilang - Bima) = (-50) + 10

Rudi + Bima = -40

Akhirnya, kita dapat mencari rata-rata nilai Try Out Rudi dan Bima:

(Rudi + Bima) / 2 = (-40) / 2 = -20

Jadi, rata-rata nilai Try Out Rudi dan Bima adalah -20.
