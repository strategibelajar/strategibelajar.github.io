19. (cos 45) / (sec 30+cosec 30)


Kategori Soal : Matematika - Trigonometri
Kelas : X (1 SMA)
Pembahasan :
Sudut-sudut istimewa adalah sudut-sudut dengan nilai perbandingan trigonometri yang dapat ditentukan tanpa menggunakan kalkulator atau tabel matematika.
Sudut-sudut istimewa yang dipaparkan dalam tabel berikut besarnya 0°, 30°, 45°, 60°, dan 90°.
 
                  0°     30°     45°         60°        90°
sin             0      1/2      1/2√2     1/2√3     1  
cos            1       1/2√3 1/2√2     1/2          0
tan            0       1/3√3 1             √3          ∞

Mari kita lihat soal tersebut.

[tex]cos 45^o= \frac{1}{2} \sqrt{2}\\ sec30^o= \frac{1}{cos30^o}= \frac{1}{ \frac{1}{2} \sqrt{3}}= \frac{1}{ \frac{ \sqrt{3} }{2} }= \frac{2}{ \sqrt{3}}= \frac{2}{ \sqrt{3} }. \frac{ \sqrt{3} }{ \sqrt{3} }= \frac{2 \sqrt{3} }{3}\\
cosec30^o= \frac{1}{sin30^o}= \frac{1}{ \frac{1}{2} }=2 [/tex]

Jadi,

[tex]\frac{cos45^o}{sec30^o+cosec30^o}\\= \frac{\frac{1}{2} \sqrt{2}}{\frac{2}{3} \sqrt{3}+2}\\= \frac{ \frac{1}{2} \sqrt{2} }{ \frac{2 \sqrt{3}+6}{3} }\\= \frac{3. \frac{1}{2} \sqrt{2}}{2 \sqrt{3}+6}\\= \frac{3 \sqrt{2} }{2(2 \sqrt{3}+6)}\\= \frac{3 \sqrt{2}}{4 \sqrt{3}+12} [/tex]

21. b. cotan² 60° + cotan² 45° + cotan² 30° - cosec² 60° - cosec² 45° - cosec² 30° =? Tolong bantu ya kak


To solve this problem, we can use the trigonometric identities:

cot²θ + 1 = csc²θ

cotθ = 1/tanθ

Using these identities, we can rewrite each term in the equation as follows:

cot² 60° = (1/tan 60°)² = (1/√3)² = 1/3

cot² 45° = (1/tan 45°)² = 1² = 1

cot² 30° = (1/tan 30°)² = (√3/3)² = 1/3

cosec² 60° = (1/sin 60°)² = (2/√3)² = 4/3

cosec² 45° = (1/sin 45°)² = 2² = 4

cosec² 30° = (1/sin 30°)² = (2/1)² = 4

Substituting these values into the equation, we get:

1/3 + 1 + 1/3 - 4/3 - 4 - 4 = -11

Therefore, cotan² 60° + cotan² 45° + cotan² 30° - cosec² 60° - cosec² 45° - cosec² 30° = -11.
