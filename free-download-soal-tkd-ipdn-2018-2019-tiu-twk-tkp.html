17. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

27. Jika x adalah 3,87 persen dari 2475 dan y adalah 4,75 persen dari 3870 makaNote:ini Soal TIU​


Jawaban:

Nilai y adalah 184.425.

Penjelasan dengan langkah-langkah:

Jika x adalah 3,87 persen dari 2475, kita dapat menghitung nilai x dengan mengalikan persentase tersebut dengan nilai 2475:

x = 3.87% x 2475

x = (3.87 / 100) x 2475

x = 0.0387 x 2475

x = 95.7225

Jadi, nilai x adalah 95.7225.

Selanjutnya, jika y adalah 4,75 persen dari 3870, kita dapat menghitung nilai y dengan cara yang sama:

y = 4.75% x 3870

y = (4.75 / 100) x 3870

y = 0.0475 x 3870

y = 184.425

Jadi, nilai y adalah 184.425.
