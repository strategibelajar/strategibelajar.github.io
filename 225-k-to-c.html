15. konversikanlah ( ubahlah ) 1. 225 k ke ..... °C2. 45 °C ke ..... °R3. 36 °R ke ..... °F4. 14 °F ke ......°R5. 14 °F ke ..... K6. 225 K ke .....°R​


Soal No 1

Diketahui untuk mengubah ke celcius, maka besar suhu kelvin harus dikurangi 273 derajat. Jadi, 225 K = (225 - 273)°C = -48°C

Soal No 2

Diketahui untuk mengubah ke reamur, maka besar suhu reamur harus dikalikan 4/5. Jadi, 45°C = 4/5 x 45 = 36°R

Soal No 3

Diketahui untuk mengubah ke fahrenheit, maka besar suhu reamur harus dikalikan 9/4 dan ditambah 32 derajat. Jadi, 36°R = 9/4 x 36 + 32 = 81 + 32 = 113°R

Soal No 4

Diketahui untuk mengubah ke fahrenheit, maka besar suhu fahrenheit harus dikurangi 32 derajat lalu dikalikan 4/9. Jadi, 14°F = (14 - 32) x 4/9 = (-18) x 4/9 = -8°R.

Soal No 5

Diketahui untuk mengubah ke Kelvin, fahrenheit harus diubah ke celcius terlebih dahulu. Untuk mengubah ke celcius, besar suhu fahrenheit harus dikurangi 32 derajat lalu dikalikan 5/9. Jadi, 14°F = (14 - 32) x 5/9 = (-18) x 5/9 = -10°C. Kemudian, besar suhu celcius harus ditambah 273 derajat untuk menjadi suhu Kelvin. Maka, -10°C = -10 + 273 = 263 K.

Soal No 6

Diketahui untuk mengubah ke reamur, Kelvin harus diubah ke celcius terlebih dahulu. Untuk mengubah ke celcius, besar suhu Kelvin harus dikurangi 273 derajat terlebih dahulu. Jadi, 225 K = 225 - 273 = -48°C. Kemudian, untuk mengubah ke reamur, besar sudut celcius harus dikalikan 4/5. Jadi, -48°C = 4/5 x (-48) = -38,4°C.
