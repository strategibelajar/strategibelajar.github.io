5. a.harga awal 100.000 diskon 20% berapa harga diskon nya b.harga awal 150.000 diskon 30% berapa harga diskon nya c.harga awal 200.000 harga diskon 150.000 berapa diskon nya d.diskon 15% harga diskon 102.000 berapa harga awalnya e.harga awal 160.000 harga diskon 144.000 berapa diskon nya


Jawaban:

a. 80.000

b. 105.000

c. 25%

d. 120.000

e. 10%

Pembahasan:Aritmetika Sosial

Aritmetika Sosial adalah persoalan matematika yang berhubungan dengan kehidupan sehari hari. Beberapa diantaranya perhitungan jual, beli, untung dan rugi, perhitungan berat kemasan dan isi, perhitungan pinjaman dan menyimpan uang berbunga, dan perhitungan potongan harga.

Potongan harga /rabat/diskon

Rumus rumua yang digunakan untuk permasalahan diskon antara lain

harga akhir = harga awal - diskonbesar diskon = prosentase diskon x harga awalprosentase diskon = [tex] \frac{besar diskon}{harga awal} [/tex] x 100%

Pembahasan Soal

a. Diketahui:

harga awal =  100.000 diskon = 20%

Ditanya: harga setelah diskon

Jawab

Langkah 1: tentukan dahulu besar diskon

besar diskon

= prosentase diskon x harga awal

= 20% x 100.000  

= [tex] \frac{20}{100} [/tex] x 100.000

= 20 x 1000

= 20.000

Langkah 2: Menentukan harga setelah diskon

harga setelah diskon

= harga awal - besar diskon

= 100.000 - 20.000

= 80.000

b. Diketahui:

harga awal = 150.000diskon = 30%

Ditanya: harga setelah diskon

Jawab

Langkah 1: tentukan dahulu besar diskon

besar diskon

= prosentase diskon x harga awal

= 30% x 150.000  

=[tex] \frac{30}{100} [/tex] x 150.000

= 30 x 1500

= 45.000

Langkah 2: Menentukan harga setelah diskon

harga setelah diskon

= harga awal - besar diskon

= 150.000 - 45.000

= 105.000

c. Diketahui:

harga awal =  200.000 harga setelah diskon 150.000

Ditanya: prosentase diskon

Jawab:

Langkah 1: tentukan dahulu besar diskon

diskon

= harga awal - harga setelah diskon

= 200.000 - 150.000

= 50.000

Langkah 2: Menentukan prosentase diskon

prosentase diskon

= [tex] \frac{besar diakon}{harga awal} [/tex] x 100%

= [tex] \frac{50.000}{200.000} [/tex] x 100 %

= [tex] \frac{1}{4} [/tex] x 100%

= 25%

d. Diketahui:

diskon = 15% harga setelah diskon = 102.000

Ditanya: harga awalnya (y)

Jawab:

Langkah 1: tentukan dahulu besar diskon

diskon

= prosentase diskon x harga awal

= 15% x y

= 15% y

Langkah 2: Menentukan harga awal

harga setelah diskon = harga awal - diskon

102.000 = y - 15% y

102.000 = 100% y - 15% y

102.000 = 85% y

y = 102.000: 85%

y = 102.000: [tex] \frac{85}{100} [/tex]

y = 102.000 x [tex] \frac{100}{85} [/tex]

y = 1200 x 100

y = 120.000

e. Diketahui:

harga awal =  160.000 harga setelah didiskon = 144.000

Ditanya: berapa prosentase diskon

Jawab:

Langkah 1: tentukan dahulu besar diskon

besar diskon

= harga awal - harga setelah didiskon

= 160.000 - 144.000

= 16.000

Langkah 2: tentukan prosesntase diskon

prosentase diskon

= [tex] \frac{besar diakon}{harga awal} [/tex] x 100%

= [tex] \frac{16.000} {160.000} [/tex] x 100 %

= [tex] \frac{1}{10} [/tex] x 100%

= 10%

semoga bermanfaat
