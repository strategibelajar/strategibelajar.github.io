18. apakah x + y = y × xx: 4,6,8,-4,-9,-13y: 5,-3,-5,-6,-11,-12x+y: ........................y+x: ..........................​


Apakah x + y = y × x?

x: 4, 6, 8, -4, -9, -13

y: 5, -3, -5, -6, -11, -12

Mari kita hitung untuk setiap nilai x dan y:

1. x = 4, y = 5

x + y = 4 + 5 = 9

y + x = 5 + 4 = 9

2. x = 6, y = -3

x + y = 6 + (-3) = 3

y + x = (-3) + 6 = 3

3. x = 8, y = -5

x + y = 8 + (-5) = 3

y + x = (-5) + 8 = 3

4. x = -4, y = -6

x + y = (-4) + (-6) = -10

y + x = (-6) + (-4) = -10

5. x = -9, y = -11

x + y = (-9) + (-11) = -20

y + x = (-11) + (-9) = -20

6. x = -13, y = -12

x + y = (-13) + (-12) = -25

y + x = (-12) + (-13) = -25

Jadi, untuk setiap pasangan nilai x dan y yang diberikan, hasil dari x + y dan y + x selalu sama.
