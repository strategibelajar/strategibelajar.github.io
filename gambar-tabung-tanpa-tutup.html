8. Sebuah tabung tertutup dengan jari-jari 20 cm dan tingginya 40 cm seperti gambar di bawah.Luas permukaan tabung tersebut


Jawaban:

a) Volume tabung  = 50.240 cm³

b) Luas alas tabung = 1.256 cm²

c) Luas tutup tabung = 1.256 cm²

d) Luas selimut tabung = 5.024 cm²

e) Luas permukaan tabung = 7.536 cm²

Pembahasan

Tabung adalah bangun ruang dengan sisi lengkung

Rumus-rumus yang ada pada bangun tabung

1) Luas alas tabung = π x r²

2) Luas selimut tabung = Keliling alas x tinggi

= 2 x π x r x t atau = π x d x t

Keliling alas = panjang selimut

3) Luas permukaan tabung tanpa tutup = luas alas + luas selimut

 = π x r² + 2 x π x r x t

 = π x r x (r + 2t)

4) Luas permukaan tabung dengan tutup =  luas alas + luas tutup + luas selimut

 = π x r² + π x r² + 2 x π x r x t

 = 2 x π x r² + 2 x π x r x t

 = 2 x π x r x (r + t)

5) Volume tabung = π x r² x t atau luas alas x t

r = jari-jari

d = diameter

t = tinggi

Penyelesaian Soal

Diketahui:

Jari-jari tabung = 20 cm

Tinggi tabung = 40 cm

Ditanya:

a) Volume tabung

b) Luas alas tabung

c) Luas tutup tabung

d) Luas selimut tabung

e) Luas permukaan tabung

Jawab:

a) Volume tabung = π x r² x t

                              = 3,14 x 20 cm x 20 cm x 40 cm

                              = 50.240 cm³

b) Luas alas tabung = π x r²

                                 = 3,14 x 20 cm x 20 cm

                                 = 1.256 cm²

c) Luas tutup tabung = π x r²

                                 = 3,14 x 20 cm x 20 cm

                                 = 1.256 cm²

d) Luas selimut tabung = 2 x π x r x t

                                 = 2 x 3,14 x 20 cm x 40 cm

                                 = 5.024 cm²

e) Luas permukaan tabung = 2 x π x r x (r + t)

                                 = 2 x 3,14 x 20 cm x (20 cm + 40 cm)

                                 = 7.536 cm²
