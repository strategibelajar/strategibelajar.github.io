10. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

26. bantu jawab, ini dari soal simulasi UNBK kemarin


[tex]Persamaan\: Kuadrat [/tex]

Persamaan Kuadrat 2x² + mx + 16 = 0 memiliki akar-akar p dan q, jika p = 2q, sedangkan p, q > 0
maka m = ... ?

dari persamaan kuadrat tersebut,
2x² + mx + 16 = 0
a = 2
b = m
c = 16

p × q = c/a = 8
(2q) × q = 8
2q² = 8
q = ±4, karena q > 0, maka q = 4

maka p = 2(4) = 8
sehingga
p + q = -b/a
8 + 4 = -m/2
12 = -m/2
m = -24 ✔️
