Penjelasan dengan langkah-langkah:

Jumlah Peserta = 30

Nilai rata2 = 364

Nilai total = 364 x jumlah peserta = 364 x 30 = 10.920

ditanya : nilai rata - rata tanpa orang yg terlambat

jawab :

nilai total baru = nilai total - nilai peserta terlambat

= 10.920 - 385

= 10.535

nilai rata-rata

= nilai total baru : jumlah peserta baru

= 10.535 : 29

= 363.27

kita misalkan

SKD = x

SKB = y

buat persamaannya

andre = 60x+70y=67 (Andre kurang 3 dr batas)

bayu = 50x+80y=71 (Bayu lebih 1 dr minimum)

Caca = 80x + 60y

Dengan metode eliminasi

60x+70y=67 (kali 5)

50x+80y=71 (kali 6)

...............

300x+350y = 335

300x+480y=426

-130y=-91

y=0, 7

x=0, 3

maka nilai Caca

80x+60y=80(0,3) + 60(0,7)

24+42

66
