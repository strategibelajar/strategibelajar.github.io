22. Yudha mengikuti seleksi tes masuk di sebuah perusahaan. Aturan penilaian skor dalam tes tersebut yaitu skor 4 untuk tiap jawaban soal benar, -2 untuk jawaban soal salah, dan -1 untuk soal tidak dijawab. Jumlah keseluruhan soal sebanyak 30. Jika Yudha hanya menjawab 27 soal dengan mendapatkan skor 75, tentukan banyak soal yang dijawab benar? ​


=> Skor dari Jawaban benar ( 4x )

=> - skor dari Jawaban salah 2 ( 30 - x ) - skor dari soal tidak dijawab ( x) artinya 1 yaa

Totall :

4x - 2 ( 30 - x ) - x = 75

=> Langsung kita selesaikan persamaan nya

4x - 60 + 2x - x = 75

5x - 60 = 75

5x = 75 + 60

5x = 135

x = 135 ÷ 5

x = 27

Jadi Yudha menjawab soal benar yaitu 27.

                                                                                 = 2 : 5
