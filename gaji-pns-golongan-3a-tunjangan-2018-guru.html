[tex]5000000 = \frac{4200000x + 7400000y}{x + y} \\ 50(x + y) = 42x + 74y \\ 50x + 50y = 42x + 74y \\ 8x = 24y \\ \frac{x}{y} = \frac{24}{8} = \frac{3}{1} [/tex]

Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.

28. 3. Bapak rahmat seorang PNS Golongan IV/a. Memliki NPWP 08.8-03.625.8.605.000. Pada Bulan Maret 2016 memiliki gaji pokok Rp. 4.500.000. Tunjangan istri 10%, tunjangan anak 2%, tunjangan fungsional Rp. 3.500.000, Iuran pensiun 4,75%. Status Kawin anak 1 . hitunglah Pajak Penghasilan atas gaji Bapak Rahmat.​


Subjek Pajak Penghasilan (PPh) adalah orang pribadi, warisan yang belum terbagi sebagai satu kesatuan, badan, dan bentuk usaha tetap. Dan Bapak rahmat seorang PNS Golongan IV/a dengan penghasilan selama setahun sebesar Rp. 99.915.000 dikenakan pajak sebesar 15% adalah sekitar Rp. 14.987.250 (PPh)

Pembahasan:

Pajak Penghasilan (PPh) adalah pajak yang dibayarkan atas penghasilan berupa gaji, upah, honorarium, tunjangan, dan pembayaran lain sehubungan dengan pekerjaan, jasa, atau kegiatan apa pun yang diterima.

Dalam Undang-Undang Nomor 36 Tahun 2008 tentang Pajak Penghasilan (PPh), subjek pajak yang menerima atau memperoleh penghasilan disebut Wajib Pajak. Wajib pajak dikenai pajak atas penghasilan yang dterima atau diperolehnya selama satu tahun pajak atau dapat pula dikenakan pajak untuk penghasilan dalam bagian tahun pajak apabila kewajiban pajak subjektifnya dimulai atau berakhir dalam tahun pajak.

Ketentuan tarif pajak penghasilan perorangan atau pribadi berdasarkan Undang-Undang Nomor 36 Tahun 2008 yang berlaku tanggal 1 Januari 2009 sebagai berikut:

Penghasilan sampai dengan Rp50 juta dikenakan pajak 5%.Penghasilan Rp50 juta–Rp250 juta dikenakan pajak 15%.Penghasilan Rp250 juta–Rp500 juta dikenakan pajak 20%.Penghasilan diatas Rp500 juta dikenakan pajak 30%

Menghitung Pajak Penghasilan (PPh) pribadi:

Pajak Penghasilan (PPh) = Penghasilan Kena Pajak (PKP) x TarifPenghasilan Kena Pajak (PKP) = Penghasilan Bersih - Penghasilan Tidak Kena Pajak (PTKP)

Diketahui:

Bapak rahmat seorang PNS Golongan IV/a gaji pokok Rp. 4.500.000Tunjangan istri 10% = Rp 450.000tunjangan anak 2% = Rp 90.000tunjangan fungsional Rp. 3.500.000Iuran pensiun 4,75% = Rp 213.750

Ditanya:

Pajak Penghasilan atas gaji Bapak Rahmat?

Jawab:

Perhitungan gaji pak rahmat

= gaji pokok + Tunjangan istri 10% + tunjangan anak 2% + tunjangan fungsional - Iuran pensiun 4,75%

= Rp 4.500.000 + Rp 450.000 + Rp 90.000 + Rp 3.500.000 - Rp 213.750

= Rp. 8.326.250

Penghasilan selama setahun = Rp. 8.326.250 x 12 = Rp. 99.915.000

Penghasilan Rp50 juta–Rp250 juta dikenakan pajak 15%.

Jadi Pajak penghasilan (PPh) pak rahmat adalah Rp. 99.915.000 x 15% = Rp. 14.987.250

39. Kartikasari, s.H. Seorang golongan pns III/b memiliki gaji pokok 3.500.000,tunjangan istri 10% tunjangan anak 2%per anak ,tunjanan fungaional 2.000.000,iuran pengsiun 4,75%.Status kawin denan 2 anak jumlah penghasilan neto kartikasari per bulan


PENGHASILAN NETTO

Penghasilan bruto (gaji pokok + tunjangan-tunjangan) yang sudah dikurangi dengan biaya-biaya (biaya jabatan, iuran pensiun, iuran BPJS, iuran asuransi, dan lain-lain)

________________________________

Langkah-Langkahnya

1. Mencari gaji pokok

2. Mencari tunjangan istri

3. Mencari tunjangan 2 anak

4. Mencari tunjangan fungsional

5. Mencari iuran pensiun

6. Mencari biaya jabatan

7. Mencari penghasilan netto

GAJI POKOK

= 3.500.000

TUNJANGAN ISTRI

= 10% x gaji pokok

= 10% x 3.500.000

= 350.000

TUNJANGAN 2 ANAK

= 2 x (2% x gaji pokok)

= 2 x (2% x 3.500.000)

= 2 x 70.000

= 140.000

TUNJANGAN FUNGSIONAL

= 2.000.000

IURAN PENSIUN

= 4,75% x gaji pokok

= 4,75% x 3.500.000

= 166.250

BIAYA JABATAN

= 5% x gaji pokok

= 5% x 3.500.000

= 175.000

PENGHASILAN NETTO

= (gaji pokok + tunjangan istri + tunjangan 2 anak + tunjangan fungsional) - (iuran pensiun + biaya jabatan)

= (3.500.000 + 350.000 + 140.000 + 2.000.000) - (166.250 + 175.000)

= 5.990.000 - 341.250

= Rp5.648.750

Jadi, penghasilan netto Kartika Sari per bulannya adalah Rp5.648.000

________________________________

BEA JABATAN adalah biaya yang dikenakan kepada wajib pajak yang memperoleh jabatan tertentu di suatu instansi, baik swasta maupun pemerintahan. Jumlahnya yakni 5% dari gaji pokok. Dengan batas maksimal pengurangan yakni Rp500.000 per bulan dan Rp6.000.000 per tahunnya
