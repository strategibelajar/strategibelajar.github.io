10. tentukan gradien dari 4x-5y-20=0


m = - a/b
m = -4/-5
m = 4/5
persamaan untuk mencari nilai gradien
[tex]m = \frac{ y_{2}-y_{1}}{x_{2}-x_{1}} [/tex]

 dari persamaan 4x - 5y - 20 = 0
jika, x₁ = 0 maka y₁ = ?
4.0 - 5y₁ - 20 = 0 ⇔ -5y₁ = 20 ⇔ y₁ = -4
jika y₂ = 0  maka x₂ = ?
4x₂ - 5.0 - 20 = 0 ⇔ 4x₂ = 20 ⇔ x₂ = 5

maka nilai gradien dari persamaan tersebut sebesar
[tex]m = \frac{ y_{2}-y_{1}}{x_{2}-x_{1}} \\ m = \frac{ 0 - (-4)}{5-0} \\ m= \frac{4}{5} [/tex]
