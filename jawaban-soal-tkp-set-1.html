6. Dari main memori seperti soal tiga diberikan masukan data : 50234E cari dengan tag dan word? Jika digunakan pemetaan set asosiatif (Set-Assosiatif Mapping) tunjukkan masing-masing berapa bit untuk tag, set dan word. Buatlah dalam: a. two-way set b. four-way set Dari main memori seperti soal satu diberikan masukan data : 20254E cari dengan 2-way set ? Dari main memori seperti soal satu diberikan masukan data : 50234E cari dengan 4-way set ?


Jawaban:

Untuk two-way set, membagi cache menjadi 2 set:Bit untuk tag = 19 bitBit untuk set = 1 bitBit untuk word = 12 bit

2. Untuk four-way set, membagi cache

menjadi 4 set:

Bit untuk tag = 18 bitBit untuk set = 2 bitBit untuk word = 12 bit

Untuk masukan data 20254E dengan two-way set, maka:Tag = 20254Set = 0 (dari bit terakhir alamat 2)Word = E

Sehingga data tersebut dapat ditemukan di dalam set 0 dengan tag 20254 dan word E.

2. Untuk masukan data 50234E dengan four-way set, maka:

Tag = 50234Set = 2 (dari dua bit terakhir alamat 5)Word = E

Sehingga data tersebut dapat ditemukan di dalam set 2 dengan tag 50234 dan word E.

33. Jumlah set dalam permainan bulu tangkis adalah.... a. 1 dan 3 set; b. 2 dan 3 set; c. 3 dan 4 set; d. 4 dan 6 set; e. 3 dan 5 set.


Dalam permainan bulu tangkis, setiap pertandingan terdiri dari (B) 2 dan 3 set. Set adalah jumlah babak dalam satu pertandingan bulu tangkis.

Pembahasan

Permainan bulu tangkis adalah salah satu olahraga bola besar yang dapat dimainkan secara tunggal atau beregu. Tujuan dari permainan bulu tangkis adalah untuk mematikan bola di area permainan lawan dan memenangkan pertandingan. Permainan bulu tangkis dimainkan di lapangan tertutup yang terhindar dari hembusan angin.

Dalam permainan bulu tangkis, setiap pertandingan dimainkan dalam 2 atau 3 babak. Jika pemain atau regu berhasil memenangkan 2 set awal, maka dinyatakan sebagai pemenang dan pertandingan berhasil. Namun jika kedua pemain atau regu imbang di 2 set awal, maka pertandingan dilanjutkan pada set ke 3 sebagai set penentuan.

Dalam permainan bulu tangkis, terdapat beberapa teknik dasar yang harus dikuasai oleh para pemain, yakni.
