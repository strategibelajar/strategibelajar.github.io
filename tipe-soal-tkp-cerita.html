2. pada soal matematika tipe peluang. bagaiman cara menentukan rumus permutasi dengan kombinasi pada soal cerita?


permutasi = persoalan menyusun
kombinasi = persoalan memilih

untuk menyusun kita harus memilih.

contoh:
(1) 3 orang dipilih dari 5 orang --> persoalan kombinasi 5C3
(2) 3 orang dipilih dari 5 orang kemudian disusun menjadi Ketua, Sekretaris, Bendahara ---> persoalan Permutasi 5P3

pada persoalan (2)
3 orang dipilih dari 5, bisa dihitung dengan kombinasi 5C3.
Kemudian 3 orang ini disusun/ditempatkan sebagai ketua, sekretaris, bendahara, karena ada 3 tempat berarti banyaknya 3!.
Jadi proses menyusun adalah 5C3 x 3! = 5P3

[tex]_5C_3=\dfrac{5!}{3!(5-3)!}=10[/tex]
[tex]3!=3\cdot 2\cdot 1=6[/tex]
[tex]_5P_3=\dfrac{5!}{(5-3)!}=60[/tex]

6. sebutkan 4 tipe sudut pandang cerita


1.    Sudut Pandang Orang Pertama Tunggal.

Penulis sebagai pelaku sekaligus narator yang menggunakan kata ganti “aku’.

2.    Sudut Pandang Orang Pertama Jamak

Ini mirip dengan Sudut Pandang Orang Pertama Tunggal, hanya saja menggunakan kata ganti “kami”. Narator menjadi seseorang dalam cerita yang bicara mewakili beberapa orang atau sekelompok orang.

Penulis ada di luar cerita tak terlibat dalam cerita. Penulis juga menampilkan para tokoh dengan menyebut namanya atau kata ganti “dia”.

Penulis adalah narator yang sedang berbicara kepada kata ganti “kamu” dan menggambarkan apa yang dilakukan “kamu” atau “kau” atau “anda”.

17. 1). setiap ujian di haruskan menjawab 20 soal tipe A dan tipe B. soal tipe A harus di kerjakan sedikitnya 3 soal dan tidak lebih dari 12 soal, dan soal tipe harus di kerjakan paling sedikit 5 soal dan tidak lebih dari 15 soal. jawaban benar soal tipe A di berikan skor 4 dan jawaban benar soal tipe B diberi skor 6. jikia amir salah seorang peserta tes tersebut maka skor maksimum yang dapat di capai amir adlah.


diketahui
[tex]A+B=20 \\\\ 3\leq A\leq 12 \\ 5\leq B\leq 15[/tex]

karena point B lebih besar dari point A, maka pilih
[tex]B=15[/tex]
yang merupakan batas atas dari syarat B agar poinnya maksimal,
maka
[tex]A+B=20 \\ A+15=20 \\ A=5[/tex]

sehingga komposisi soal nya adalah
5 soal tipe A dan 15 soal tipe B
dengan total poin yang didapat
[tex](5\times 4)+(15\times 6)=20+90=110[/tex]

18. Mohon bantuannya Tipe soal tentang apakah ini?


Tipe Soal Perbandingan

Penyelesaian :
Misal, kran pertama = A
kran kedua = B
kran ketiga = C

Diketahui, A+B = 70 menit
A+C = 84 menit
B+C = 140 menit

Ditanyakan : waktu tercepat = ?
Waktu tercepat dapat dicapai apabila ketiga kran bekerja bersama atau A+B+C.
A+B+C = ½[(A+B) + (B+C) + (A+C)]

A+B = 70 menit
A+B dalam 1 menit dapat menyelesaikan 1/70 bagian. Pada perhitungan ini kita pakai yang 1/70, bukan 70. Sehingga :
A+B+C = ½[(1/70) + (1/84) + (1/140)]
A+B+C = ½[1/30]
A+B+C = 1/60

A+B+C dalam 1 menit mampu menyelesaikan 1/60 pekerjaannya, maka A+B+C akan menyelesaikan pekerjaannya dalam waktu 60 menit (Opsi A).

// misal setiap bagian tandon yang dapat di kosongkan setiap menitnya oleh kran pertama adalah a

   misal setiap bagian tandon yang dapat di kosongkan setiap menitnya oleh kran kedua adalah b

   misal setiap bagian tandon yang dapat di kosongkan setiap menitnya oleh kran ketiga adalah c //


// karena waktu yang dibutuhkan untuk 1 pekerjaan tersebut setiap menitnya = 1 / d //


// mencari b - c dengan eliminasi //

a + b = 1 / 70 menit

a + c = 1 / 84 menit

____________-

  b - c = 1 / 70 menit - 1 / 84 menit

          = 6 / 420 menit - 5 / 420 menit

          = 1 / 420 menit


// mencari c dengan eliminasi //

 b - c = 1 / 420 menit

b + c = 1 / 140 menit

_____________-

-2c = 1 / 420 menit - 1 / 140 menit

   c = 1 / 420 menit - 3 / 420 menit/ -2

   c = -2 / 420 menit × 1 / -2

   c = 2 / 420 menit × 1 / 2

   c = 1 / 420 menit


// mencari b dengan mensubstitusikan c //

                     b - c = 1 / 420 menit

b - (1 / 420 menit) = 1 / 420 menit

                           b = 1 / 420 menit + 1 / 420 menit

                           b = 2 / 420 menit

                           b = 1 / 210 menit


// mencari a dengan mensubstitusikan b //

a + b = 1 / 70 menit

      a = 1 / 70 menit - 1 / 210 menit

      a = 3 / 210 menit - 1 / 210 menit

      a = 2 / 210 menit

      a = 1 / 105 menit


// mencari waktu tercepat dengan menambahkan a + b+ c //

1 / waktu tercepat = a + b + c                                      // ketiga kran hidup //

1 / waktu tercepat = 1 / 105 menit + 1 / 210 menit + 1 / 420 menit

1 / waktu tercepat = 4 / 420 menit + 2 / 420 menit + 1 / 420 menit

1 / waktu tercepat = 7 / 420 menit

1 / waktu tercepat = 1 / 60 menit

     waktu tercepat = 60 menit


// Pembahasan : waktu tercepat adalah waktu yang dibutuhkan untuk mengosongkan tandon, sedangkan 1 / waktu tercepat adalah setiap bagian dari tandon yang kosong setiap menitnya //


// Jadi dalam waktu 60menit ketiga kran tersebut dapat mengosongkan 1 tandon bila hidup bersamaan, Jawabannya A //

37. setiap peserta ujian diharuskan menjawab 20 soal tipe a dan tipe b. soal tipe a harus dikerjakan sedikitnya 3 soal dan tidak lebih dari 12 soal, sedangkan soal tipe b harus dikerjakan paling sedikitnya 5 soal dan tidak lebih dari 15 soal. jawaban benar untuk soal tipe a diberi skor 4 dan jawaban benar untuk soal tipe b diberi skor 6. jika popon salah seorang peserta ujian tersebut tentukan skor maksimum yang dapat dicapainya?


Setiap peserta ujian diharuskan menjawab 20 soal tipe A dan tipe B. Soal tipe A harus dikerjakan sedikitnya 3 soal dan tidak lebih dari 12 soal, sedangkan soal tipe B harus dikerjakan paling sedikitnya 5 soal dan tidak lebih dari 15 soal. Jawaban benar untuk soal tipe A diberi skor 4 dan jawaban benar untuk soal tipe B diberi skor 6. Jika popon salah seorang peserta ujian tersebut tentukan skor maksimum yang dapat dicapainya?

Pembahasan :

Misal
x = soal tipe A
y = soal tipe B

• Peserta ujian diharuskan menjawab 20 soal
x + y ≤ 20

• Soal tipe A harus dikerjakan sedikitnya 3 soal dan tidak lebih dari 12 soal
3 ≤ x ≤ 12

• Soal tipe B harus dikerjakan sedikitnya 5 soal dan tidak lebih dari 15 soal
5 ≤ y ≤ 15

• Skor benar soal tipe A = 4 dan skor benar soal tipe B = 6 => fungsi sasaran
f(x, y) = 4x + 6y

Setelah kita gambar grafiknya (bisa dilihat di lampiran) maka diperoleh titik-titik sudutnya yaitu A(3, 5), B(12, 5), C(12, ...), D(..., 15) dan E(3, 15)

Untuk mencari titik C kita substitusikan x = 12 ke
x + y = 20
12 + y = 20
y = 8
C(12, 8)

Untuk mencari titik D kita substitusikan y = 15 ke
x + y = 20
x + 15 = 20
x = 5
D(5, 15)

Setelah itu kita substitusikan kelima titik tersebut ke fungsi sasaran
f(x, y) = 4x + 6y
A(3, 5) => 4(3) + 6(5) = 12 + 30 = 42
B(12, 5) => 4(12) + 6(5) = 48 + 30 = 78
C(12, 8) => 4(12) + 6(8) = 48 + 48 = 96
D(5, 15) => 4(5) + 6(15) = 20 + 90 = 110
E(3, 15) => 4(3) + 6(15) = 12 + 90 = 102

Jadi skor maksimum yang bisa dicapai adalah 110 (5 soal tipe A dan 15 soal tipe B)
