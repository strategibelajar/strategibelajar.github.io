3. TOLONG BUAT PROGRAM PYTHON Perhitungan Biaya Rawat Inap Nomor Pasien : Input Nama Pasien : Input Nomor Kamar : Input Kelas Kamar : Input Lama Inap : Input Biaya Inap per hari : Output Total Biaya Inap : Output Layanan Injeksi (10%) : Output Layanan Kebersihan (5%) : Output Layanan Laundry (8%) : Output Total Bayar : Output Mau Input Lagi [y/T] : Input *Ketentuan Proses: Biaya inap per hari - Kelas Ekonomi --> 150.000 - Kelas Standar --> 200.000 - Kelas Eksekutif --> 300.000


Jawaban:

#Perhitungan Biaya Rawat Inap

while True:

nomor_pasien = input("Nomor Pasien: ")

nama_pasien = input("Nama Pasien: ")

nomor_kamar = input("Nomor Kamar: ")

kelas_kamar = input("Kelas Kamar (Ekonomi/Standar/Eksekutif): ")

lama_inap = int(input("Lama Inap (dalam hari): "))

#Menentukan biaya inap per hari berdasarkan kelas kamar

if kelas_kamar.lower() == "ekonomi":

biaya_inap_per_hari = 150000

elif kelas_kamar.lower() == "standar":

biaya_inap_per_hari = 200000

elif kelas_kamar.lower() == "eksekutif":

biaya_inap_per_hari = 300000

else:

print("Kelas Kamar tidak valid")

break

#Menghitung total biaya inap

total_biaya_inap = biaya_inap_per_hari * lama_inap

#Menghitung biaya layanan tambahan

layanan_injeksi = total_biaya_inap * 0.1

layanan_kebersihan = total_biaya_inap * 0.05

layanan_laundry = total_biaya_inap * 0.08

#Menghitung total bayar

total_bayar = total_biaya_inap + layanan_injeksi + layanan_kebersihan + layanan_laundry

#Menampilkan output

print("Biaya Inap per hari: Rp", biaya_inap_per_hari)

print("Total Biaya Inap: Rp", total_biaya_inap)

print("Layanan Injeksi (10%): Rp", layanan_injeksi)

print("Layanan Kebersihan (5%): Rp", layanan_kebersihan)

print("Layanan Laundry (8%): Rp", layanan_laundry)

print("Total Bayar: Rp", total_bayar)

#Menanyakan apakah ingin input lagi

lagi = input("Mau Input Lagi? (y/T) ")

if lagi.lower() != "y":

break

35. Rombongan Tour kelas XII SMA “XYZ” terdiri dari 36 siswa. Mereka mengadakan wisata ke sebuah tempat dan menginap di hotel. Hotel tersebut menyediakan paling sedikit 10 kamar yang disewakan. Ada 2 tipe kamar yang dapat disewa yaitu kamar tipe I dan kamar tipe II. Kamar tipe I dapat di tempati 3 siswa dengan biaya sewa tiap harinya Rp300.000,00, sedangkan kamar tipe II dapat ditempati untuk 4 siswa dengan biaya sewa tiap harinya Rp360.000,00. Mereka menginginkan biaya sewa hotel minimal. Dengan demikian banyak kamar yang di sewa adalah ….


Misal
x = banyaknya kamar tipe I
y = banyaknya kamar tipe II

Langkah langkah :

Menentukan titik pojok
- titik potong dengan sumbu x (y=0) dan y(x=0)

x + y ⩾ 10 (10,0) (0,10)
3x + 4y ⩾ 36 (12,0) (0,9)

Karena kedua pertidaksamaan bertanda lebih dari, ambil koordinat dengan nilai x terbesar dari titik potong terhadap sumbu x, begitu juga dengan sumbu y.
(12,0) (0,10)

- titik potong grafik
SPLDV

x + y = 10 (kali 3)
3x + 4y = 36 (kali 1)

3x + 3y = 30
3x + 4y = 36
______________-
-y = - 6
y = 6

x + y = 10
x = 10 - y
= 10 - 6
= 4
(4,6)

Titik pojok = {(0,10), (4,6), (12,0)}


- Menentukan nilai optimum (minimum)
P(x, y) = Ax + By

P(0,10) = (300.000x0) + (360.000x10) = 3.600.000

P(4,6) = (300.000x4) + (360.000x6) = 3.360.000 [nilai minimum]

P(12,0) = (300.000x12) + (360.000x0) = 3.600.000

Jadi, banyaknya kamar yang disewa adalah 4 kamar tipe I dan 6 kamar tipe II. Total 10 kamar.

Jawaban:


Jika mengacu pada rumus matematika  nya :
Rumus :

Jumlah bayar = (tarif/malam x jumlah malam) + biaya inap

Artinya  :

Tarif/malam : harga sewa kamar per malamJumlah malam : durasi menginap dalam kamar hotelBiaya inap : biaya tambahan seperti pajak atau biaya layanan (klo enggak ada ditulis 0 aja)


Flowchart :

MulaiInput tarif/malam  Input jumlah malam Input biaya inap Hitung jumlah bayar  : jumlah_bayar = (tarif_per_malam* jumlah_malam)+ biaya_inapTampilkan jumlah bayarSelesai

Jika mengacu pada pemrograman nya :

Bahasa C++ :
#include <iostream>

using namespace std;

int main() {

  int tarif_per_malam, jumlah_malam, biaya_inap, jumlah_bayar;

  cout << "Masukkan tarif/malam: ";

  cin >> tarif_per_malam;

  cout << "Masukkan jumlah malam: ";

  cin >> jumlah_malam;

  cout << "Masukkan biaya inap: ";

  cin >> biaya_inap;

  jumlah_bayar = (tarif_per_malam * jumlah_malam) + biaya_inap;

  cout << "Jumlah bayar: " << jumlah_bayar << endl;

  return 0;

}
