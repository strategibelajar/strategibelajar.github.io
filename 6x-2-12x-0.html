Soal tersebut merupakan soal matematika yang membahas tentang persamaan kuadrat. Persamaan kuadrat merupakan persamaan polinominal (bersuku banyak) dengan pangkat tertingginya adalah pangkat 2.

Ditanyakan:

Tentukan akar-akar dari persamaan kuadrat tersebut!

15. Himpunan penyelesaian dari 6x² - 12x = 0 adalah​


Himpunan penyelesaian dari 6x² - 12x = 0 adalah

Hp = {0 , 2}

Pembahasan :

Persamaan kuadrat adalah persamaan suku banyak (polinomial) satu variabel yang memiliki pangkat tertinggi dua

Bentuk umum persamaan kuadrat adalah

[tex]\boxed{ax^2 + bx + c = 0}[/tex]

Dengan a, b, c adalah anggota bilangan real dan a ≠ 0

Cara menyelesaian persamaan kuadrat

1. Faktorisasi

[tex]ax^2 + bx + c = 0[/tex]

[tex]\iff \: x^2 + \dfrac{b}{a}x + \dfrac{c}{a} = 0[/tex]

Cari dua bilangan yang apabila dijumlahkan hasilnya [tex]\frac{b}{a}[/tex] dan bila dikalikan hasilnya [tex]\frac{c}{a}[/tex]

2. Kuadrat sempurna

Bentuk kuadrat sempurna adalah

[tex]\boxed{{(x + p)}^2 = x^2 + 2px + p^2}[/tex]

3. Rumus ABC

[tex]\boxed{x_{1,2} = \dfrac{-b \pm \sqrt{{b}^{2} - 4ac}}{2a}}[/tex]

Pertanyaan : Himpunan penyelesaian dari 6x² - 12x = 0

Penyelesaian :

Cara yang paling mudah dan cepat untuk menyelesaikan persamaan kuadrat di atas adalah dengan cara memfaktorkan

[tex]6 {x}^{2} - 12x = 0 \\ \\ 6x(x - 2) = 0 \\ \\ 6x = 0 \: \: atau \: \: x - 2 = 0 \\ \\ x = 0 \: \: atau \: \: x = 2[/tex]

32. limit x->0 12x⁵-2³x+5/6x⁵+3x²-2​


JAWABAN

limit x->0 12x⁵-2³x+5/6x⁵+3x²-2

Limit dari ekspresi tersebut saat x mendekati 0 adalah -2.5.

PEMBAHASAN

Untuk mencari limit ketika x mendekati 0 dari ekspresi (12x⁵ - 2³x + 5) / (6x⁵ + 3x² - 2), kita perlu memperhatikan suku-suku yang mendominasi saat x mendekati 0.

Pada suku-suku tertentu dalam ekspresi tersebut, kita melihat bahwa pangkat tertinggi dari x adalah ⁵. Ketika x mendekati 0, suku-suku yang mengandung x⁵ akan mendominasi dari suku-suku lainnya, sehingga kita dapat memfokuskan pada suku-suku ini dalam mencari limit.

Dalam pembilang, suku-suku yang mengandung x⁵ adalah 12x⁵ dan -2³x. Keduanya akan mendekati nol saat x mendekati 0, karena pangkat tertinggi dari x adalah ⁵. Sementara itu, suku konstan 5 tidak berubah saat x mendekati 0. Jadi, limit pembilang adalah 5.

Pada penyebut, suku-suku yang mengandung x⁵ adalah 6x⁵ dan suku konstan -2. Ketika x mendekati 0, suku-suku ini juga akan mendekati nol, sementara suku yang mengandung x² yakni 3x² hanya memberikan kontribusi kecil pada limit saat x mendekati 0. Jadi, limit penyebut adalah -2.

Dengan demikian, limit keseluruhan adalah limit pembilang (5) dibagi dengan limit penyebut (-2), yang memberikan hasil -2.5.

Jadi, limit dari ekspresi (12x⁵ - 2³x + 5) / (6x⁵ + 3x² - 2) saat x mendekati 0 adalah -2.5.
