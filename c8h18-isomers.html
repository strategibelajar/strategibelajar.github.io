24. Berapa mol karbon(C) dalam 3gram C8H18?


dari tabel periodik didapatkan:
Ar C = 12
Ar H = 1
Mr C8H18 = 114

massa C = (Ar C / Mr C8H8) x massa C8H8
massa C = (12 / 114) x 3
massa C = 6/19 gr

mol C = massa C / Ar C
mol C = (6/19) / 12
mol C ≈ 0,0263 mol

27. Pembakaran tidak sempurna untuk c8h18+O2


Jawaban:

Jawaban:

16,8 gram

Penjelasan:

1. dibuat dulu persamaan reaksinya setara.

2C8H18 + 25O2 --> 16CO2 + 18H2O

2. tentukan mol C8H18 dari 1,5kg

Mr C8H18 = C (12x8) + H (1x18) = A g/mol

mol C8H18 = massa / Mr

                  = 1,5 kg / A g/mol = 1500 g / A g/mol = B mol

3. tentukan jumlah mol O2 yg harus di pakai untuk menghabiskan 1.5kg C8H18

2 mol C8H18 butuh 25 mol O2, berarti 2xB = 25xmol O2

mol O2 = 2xB/25 = C mol

4. ubah mol O2 menjadi gram.

gram = mol O2 x Mr O2 = C x 16 = 16,8 gram
