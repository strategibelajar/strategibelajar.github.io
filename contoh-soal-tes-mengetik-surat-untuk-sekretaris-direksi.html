4. seorang sekretaris dapat mengetik 540 kata dalam waktu 30 menit dalam waktu 2,5 jam berapa kata yang dapat diketik sekretaris itu


Mapel : Matematika
Materi : Aritmatika Sosial
Kelas : 7

2,5 jam = 150 menit

[tex] \frac{30}{150} = \frac{540}{x} [/tex]

30x = 150 x 540
30x = 81.000
x = 81.000/30
x = 2.700 kata

Jadi, kata yang dapat diketik sekretaris itu sebanyak 2.700 kata.

150 menit = .... kata

2,5 jam = 5/2 jam

1/2 / 620 = 5/2 / x
1/2 x = 1.550
x = 1.550 / 1/2
x = 1.550 × 2/1
x = 3100 kata
