Jika Tony mendapatkan nilai 80 pada try out berikutnya, rata-rata try out Tony adalah 85. Sedangkan jika Tony mendapatkan nilai 95 pada try out berikutnya, rata-ratanya menjadi 88. Berapa banyak jumlah Try Out seluruhnya yang diikuti Tony ?


anggap aja

n = bnyak tryout

x = jmlah nilai tryout sebelumnya

berarti :

Xn + 80 / (n+1) = 85..

Xn + 80 = 85n + 85

Xn = 85n + 5...(1)

Xn + 95 /(n+1) = 88

Xn + 95 = 88n + 88

Xn = 88n - 7...(2)

eliminasi deh.

Xn = 85n + 5

Xn = 88n - 7

---------------------- -

0 = -3n + 12

-3n = -12

n = 12/3

n = 4

19. Jawaban DETIK DETIK IPA FISIKA SMP 2017 2018 Try Out 3 nomer 17 hal 179


Getaran dan Gelombang

Diketahui:
frekuensi (f) = 25000 Hz
kedalaman (s) = 15 km = 15000 m
waktu (t) = 20 s
Ditanya:
panjang gelombang (lamda) = ?

Jawab:
s = v x t : 2
15000 = v x 20 : 2
15000 x 2 : 20 = v
v = 30000 : 20
v = 1500 m/s

v = lamda x f
lamda = v : f
lamda = 1500 : 25000
lamda = 0,06 m
Jadi besar panjang gelombang ultrasonik adalah 0,06 m.

Jawaban: B
