19. Untuk membuat desain PDF itu pada di aplikasi di PC/LAPTOP apa ?​


Untuk membuat desain PDF, Anda dapat menggunakan berbagai aplikasi di PC atau laptop, tergantung pada jenis desain yang ingin Anda buat. Berikut beberapa aplikasi yang umum digunakan:

1. Adobe Acrobat Pro: Ini adalah aplikasi khusus PDF yang sangat canggih, yang dapat digunakan untuk membuat, mengedit, dan mengelola file PDF. Ini ideal jika Anda perlu membuat PDF dengan fitur-fitur lanjutan.

2. Adobe InDesign: Jika Anda ingin membuat desain yang kompleks dan kemudian mengonversinya menjadi PDF, InDesign adalah pilihan yang baik. Biasanya digunakan untuk membuat buku, majalah, brosur, dan materi desain lainnya.

3. Microsoft Word: Word memungkinkan Anda untuk membuat dokumen yang dapat Anda konversi menjadi PDF. Ini baik untuk laporan, surat, atau dokumen teks sederhana.

4. Microsoft PowerPoint: PowerPoint adalah pilihan yang baik jika Anda ingin membuat presentasi visual dan kemudian mengonversinya menjadi PDF.

5. Adobe Illustrator: Jika Anda ingin membuat vektor grafis atau ilustrasi, Illustrator adalah pilihan yang baik. Anda dapat menyimpan karya Anda dalam format PDF.

6. Canva: Canva adalah alat desain online yang dapat digunakan untuk membuat berbagai jenis desain, termasuk poster, kartu ucapan, media sosial, dan banyak lagi. Anda dapat mengunduh desain Anda dalam format PDF.

7. Sketch: Ini adalah alat desain khusus Mac yang sering digunakan untuk desain antarmuka pengguna.

8. GIMP: GIMP adalah perangkat lunak grafis gratis yang mirip dengan Adobe Photoshop. Anda dapat membuat desain dan menyimpannya dalam format PDF.

9. CorelDRAW: Ini adalah aplikasi desain vektor yang digunakan untuk membuat ilustrasi, logo, dan desain lainnya yang dapat disimpan dalam format PDF.

Pilihlah aplikasi yang paling sesuai dengan kebutuhan Anda dan jenis desain yang ingin Anda buat. Semua aplikasi di atas dapat digunakan untuk membuat desain yang dapat diubah menjadi PDF.

22. Tony telah mengikuti beberapa Try Out matematika yang diadakan oleh gurunya disekolah. Diketahui bahwa masih ada satu try out lagi yang harus diikuti Tony. Jika Tony mendapatkan nilai 80 pada try out berikutnya, rata-rata try out Tony adalah 85. Sedangkan jika Tony mendapatkan nilai 95 pada try out berikutnya, rata-ratanya menjadi 88. Berapa banyak jumlah Try Out seluruhnya yang diikuti Tony ?


anggap aja

n = bnyak tryout

x = jmlah nilai tryout sebelumnya

berarti :

Xn + 80 / (n+1) = 85..

Xn + 80 = 85n + 85

Xn = 85n + 5...(1)

Xn + 95 /(n+1) = 88

Xn + 95 = 88n + 88

Xn = 88n - 7...(2)

eliminasi deh.

Xn = 85n + 5

Xn = 88n - 7

---------------------- -

0 = -3n + 12

-3n = -12

n = 12/3

n = 4

berarti tony 4 x ngikut tryout..

38. Tolong Kakak ini untuk bahan try out


1. Bangun : Persegi Panjang
 Luas = panjang x lebar
= 25 x 12
= 300cm²
Keliling = 2 x ( panjang + lebar )
= 2 x (25 + 12)
= 2 x 37
= 74cm

2. Bangun: Segitiga
Luas = 1/2 x alas x tinggi
= 1/2 x 35 x 30
= 1/2 x 1.050
= 525cm²
Keliling = a + b + c
= 30 + 35 + 40
= 105cm

3. Bangun: Trapesium
Luas = 1/2 x (a+b) x t
= 1/2 x (25+28) x 20
= 1/2 x 53 x 20
= 1/2 x 1.060
= 530cm²
Keliling = a+b+c+d
= 25 + 28 + 20 + 30
= 103cm

4. Jajargenjang
Luas = alas x tinggi
= 28 x 30
= 840cm²
Keliling = 2 x (a+t)
= 2 x (28 + 30)
= 2 x 58
= 116cm

5. Belah Ketupat
Luas = 1/2 x d1 x d2
= 1/2 x 8 x 6
= 1/2 x 48
= 24cm
Keliling = s+s+s+s
= 8+8+6+6
= 28cm
