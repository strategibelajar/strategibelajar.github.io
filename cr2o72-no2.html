7. Pada persamaan reaksi redoks :

a Cr₂O₇²⁻ + b H⁺ + c NO₂- → 2Cr³⁺ + 3NO₃⁻ + 4H₂O

Harga koefisien reaksi a, b, dan c adalah 1, 8 dan 3

Pembahasan

Reaksi redoks atau reaksi reduksi dan oksidasi berkaitan dengan perubahan bilangan oksidasi. Setiap reaksi redoks memiliki setengah reaksi oksidasi dan setengah reaksi reduksi.

Reaksi oksidasi merupakan reaksi pengikatan oksigen, pelepasan elektron dan kenaikan bilangan oksidasi (biloks), sedangkan reaksi reduksi merupakan reaksi pelepasan oksigen, pengikatan elektron dan penurunan biloks

Penyetaraan reaksi redoks dapat disetarakan dengan dua cara yaitu metode setengah reaksi dan metode bilangan oksidasi

Suatu reaksi redoks dinyatakan setara apabila :

Jumlah atom diruas kiri sama dengan jumlah atom diruas kanan Jumlah muatan diruas kiri sama dengan jumlah muatan diruas kanan Jika reaksi berlangsung dalam kondisi asam atau basa, maka ion H⁺ dan ion OH- harus dituliskan juga

Prinsip penyetaraan reaksi redoks dengan metode setengah reaksi adalah jumlah elektron yang dilepas pada setengah reaksi oksidasi sama dengan jumlah elektron yang diikat pada setengah reaksi reduksi.  Langkah penyetaraan reaksi redoks :

Menentukan unsur-unsur  yang mengalami reaksi reduksi dan oksidasi dari perubahan biloksMenuliskan persamaan setengan reaksi yang mengalami reaksi reduksi dan oksidasi secara terpisah Menyetarakan jumlah atom unsur-unsur yang mengalami reaksi redoks pada setengah reaksi  Menyetarakan jumlah atom O dan H dengan cara : Jika kondisi reaksi asam, maka ruas yang kekurangan atom O ditambah H₂O sejumlah kekurangan atom O dan ruas yang kelebihan atom O ditambah ioh H⁺ sejumlah kekurangan atom H Jika kondisi reaksi basa, maka ruas yang kelebihan atom O ditambah H₂O sejumlah kelebihan atom O dan ruas yang kekurangan atom O ditambah ioh OH⁻ sejumlah kekurangan atom H Menyetarakan muatan kedua setengah reaksi dengan menambahkan elektron Menyetarakan jumlah elektron pada kedua setengah reaksi redoks kemudian menjumlahkan kedua reaksi tersebutMenuliskan ion-ion yang tidak terlibat dalam reaksi redoks, kemudian menjumlahkannya dan menggabungkan ion-ion tersebut menjadi molekul

Prinsip penyetaraan reaksi redoks dengan metode bilangan oksidasi adalah jumlah pertambahan bilangan oksidasi dari reduktor (yang mengalami reaksi oksidasi) sama dengan jumlah penurunan bilangan oksidasi dari oksidator (yang mengalami reaksi reduksi).  Langkah penyetaraan reaksi redoks :

Menentukan unsur-unsur  yang mengalami reaksi reduksi dan oksidasi dari perubahan biloksMenyetarakan jumlah atom unsur yang mengalami reaksi redoks Menentukan jumlah penambahan biloks dari reduktor dan jumlah penurunan biloks dari oksidatorMenyamakan jumlah perubahan bilangan oksidasi  Menyetarakan unsur lainnya dalam urutan KAHO (Kation, Anion, Hidrogen, Oksigen)

Penyelesaian Soal

Penyetaraan persamaan reaksi redoks dilakukan dengan metode setengah reaksi dalam kondisi asam (ada ion H⁺ dalam persamaan redoks)

a Cr₂O₇²⁻ + b H⁺ + c NO₂- → 2Cr³⁺ + 3NO₃⁻ + 4H₂O

Biloks Cr berubah dari +6 (dalam Cr₂O₇²⁻) menjadi +3 sehingga mengalami reaksi reduksi

Biloks N berubah dari +3 (dalam NO₂⁻) menjadi +5 (dalam NO₃⁻) sehingga mengalami reaksi oksidasi

Reduksi : 6e⁻ + 14H⁺ +Cr₂O₇²⁻ → 2Cr³⁺ + 7H₂O | dikali 1

Oksidasi : NO₂⁻ + H₂O → NO₃⁻ + 2H⁺ + 2e⁻ | dikali 3

Kedua reaksi dijumlahkan menghasilkan persamaan reaksi redoks yang setara yaitu :

Cr₂O₇²⁻ + 3NO₂⁻ + 8H⁺ → 2Cr³⁺ + 3 NO₃⁻ + 4H₂O

Catatan :

Pada reaksi reduksi:

ruas kanan kekurangan atom O ditambahkan 7 atom H₂O dan ruas kiri ditambahkan 14H⁺ sesuai jumlah atom H diruas kiri.  

Muatan ruas kiri = +12, muatan ruas kanan = +6 maka ditambahkan 6 elektron diruas kiri  

Pada reaksi oksidasi :

ruas kiri kekurangan atom O ditambahkan 1 atom H₂O dan ruas kanan ditambahkan 2H⁺  

Muatan ruas kiri = -1, muatan ruas kanan = +1 maka ditambahkan 2 elektron diruas kanan

Sedangkan reaksi reduksi merupakan reaksi penangkapan elektron, molekul atau ion. dengan kata lain, dalam reaksi reduksi terjadi pengurangan bilangan oksidasi.

10. bilangan oksidasi Cr pada ion Cr2O72- dan senyawa K2CrO4 secara berurutan adalah.....


Bilangan oksidasi Cr pada ion Cr₂O₇⁻² adalah +6. Bilangan oksidasi Cr dalam senyawa K₂CrO₄ adalah +6

Pembahasan

∴ Bilangan oksidasi ( biloks) Cr pada Cr₂O₇⁻² adalah +6.

Untuk memperoleh bilangan oksidasi ( biloks) atom Cr dapat dilakukan dengan sutitusi dari bilangan oksidasi yang sudah diketahui pada senyawa Cr₂O₇⁻²

∵Bilangan oksidasi ( biloks)  total Cr₂O₇⁻²  adalah -2 ( karena bilang oksidasi ( biloks) suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi ( biloks)  O adalah -2 ( karena bilangan oksidasi ( biloks) untuk Oksigen ( O) pada umumnya  adalah -2 )

Cr₂O₇⁻²  = -2

(jumlah atom Cr x biloks atom Cr) + (jumlah atom O x biloks atom O) = -2

(2 x biloks Cr) + (7x (-2)) = -2

2 Biloks Cr - 14 = -2

2 Biloks Cr= 14-2

2 Biloks Cr = +12

Biloks Cr = +6

∴ Bilangan oksidasi Cr pada K₂CrO₄ adalah +6.

Untuk memperoleh bilangan oksidasi ( biloks) atom Cr dapat dilakukan dengan subtitusi dari bilangan oksidasi ( biloks) yang sudah diketahui pada senyawa K₂CrO₄

∵Biloks total K₂CrO₄ adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 ) dan bilangan oksidasi  K adalah +1 ( karena bilangan oksidasi untuk unsur golongan IA ( Li, Na, K, Rb, Cs, Fr) adalah +1)

K₂CrO₄ = 0

(jumlah atom K x biloks atom K) + (jumlah atom Cr x biloks atom Cr) + (jumlah atom O x biloks atom O) = 0

(2 x 1) + (1 x biloks Cr) + (4x (-2)) = 0

2 + Biloks Cr - 8 = 0

Biloks Cr= 8-2

Biloks Cr = +6

Pembahasan

Reaksi-reaksi kimia yang tidak dapat dilihat dari pengikatan oksigen atau transfer elektron
seiring berkembangnya jaman reaksi redoks dikaitkan dengan bilangan oksidasi. Bilangan Oksidasi iti sendiri adalah bilangan yang menyatakan muatan Elektron, baik positif maupun negatif.

Untuk menjawab soal-soal seputar reaksi redoks Anda harus tau aturan-aturan redoks. terdapat beberapa aturan yang Harus diingat
sebagai berikut.

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.

Bila Sudah memahami aturan-aturan biloks Mari Kita Tentukan b.o dalam pertanyaan

19. Dalam 4,6 gram NO2 (Ar N = 14; O = 16) terdapat ... molekul NO2.Dalam 4,6 gram NO2 (Ar N = 14; O = 16) terdapat ... molekul NO2.


Dalam 4,6 gram NO₂ (Ar N = 14 ; O = 16) terdapat 6,02 × 10²² molekul NO₂

Pembahasan

Mol adalah satuan yang digunakan untuk menyatakan jumlah partikel yang terkandung dalam sejumlah zat.

Kali ini kita akan bahas, Hubungan Mol dg Massa dan Hubungan Mol dg Jumlah Partikel.

Dalam hal ini, terdapat ketetapan, yaitu bilangan Avogadro yang menyatakan 1 mol = 6,02 × 10²³ partikel.

Rumus yg digunakan pada Hubungan Mol dg Massa :

gr = n × Mr atau Ar

gr = n × Mr atau Ar n = gr / (Mr atau Ar)

Keterangan :

Massa = gr

Mol = n

Mr = massa molekul relatif

Ar = massa atom relatif

Rumus yang digunakan dalam Hubungan Mol dg Jumlah Partikel :

X = n . L

Keterangan :

X = jumlah partikel

n = mol

L = bilangan Avogadro (6,02 × 10²³)

~~~~~~~~~~~~~~~~~~

Diketahui :

gr = 4,6 gram NO₂

Ar N = 14

Ar O = 16

Ditanya :

Jumlah partikel (X) ?

Jawab :

n = gr / Mr

= 4,6 / (14 + 2 × 16)

= 4,6 / 46

= 0,1 mol

Maka,

X = n . L

= 0,1 . 6,02 × 10²³

= 0,602 × 10²³

= 6,02 × 10²² molekul ✔

Jadi, jumlah partikelnya adalah 6,02 × 10²² molekul.

27. cr2o72-+ I-=cr3++i2 basa setengah reaksi


Break up the reaction into two half-reactions -- one for Cr and one for I.
Cr2O72- -----> 2Cr3+
I- ------> IO3-

After balancing the Cr and I, balance the O by adding H2O.
Cr2O72- -----> 2Cr3+ + 7H2O
I- 3H2O ------> IO3-

Next, balance the H by adding H+
Cr2O72- + 14H+ -----> 2Cr3+ + 7H2O
I- + 3H2O ------> IO3- + 6H+

Balance the charges by adding e- to the more positive side of each half-reaction.
Cr2O72- + 14H+ + 6e- -----> 2Cr3+ + 7H2O (reduction)
I- + 3H2O ------> IO3- + 6H+ + 6e- (oxidation)

Since each half-reaction has 6e- involved, just add the half-reactions together. Simplify the water and H+.
Cr2O72- + 14H+ + 6e- -----> 2Cr3+ + 7H2O
I- + 3H2O ------> IO3- + 6H+ + 6e-

I- + Cr2O72- + 8H+ -----> 2Cr3+ + 4H2O

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0
