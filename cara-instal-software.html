3. Carilah informasi cara menginstal software!


Mapel: TI

Materi: Instalasi Software

--Soal--

Terlampir.

--Jawaban--

Dalam melakukan instalasi terhadap software, ada beberapa hal yang harus diperhatikan dan dilakukan.

1. Biasakan baca teks README atau referensi menginstal software tersebut dari sumber resminya serta pastikan developernya terpercaya.

2. Cek apakah software tersebut kompatibel dengan perangkat kita, jika tidak segera hubungi developer software tersebut untuk panduan.

3. Pastikan ruang penyimpanan disk perangkat kita tidak penuh untuk menginstall/memasang software dan requirementsnya.

4. Setelah mengetahui software tersebut kompatibel, pasang/installlah terlebih dahulu requirements software tersebut. Seperti MS Visual Studio, Net Framework, Java Runtime, dan sebagainya sesuai referensi.

5. Setelah semua siap, install/pasang software tersebut sesuai tutorial/referensi yang tersedia. Baca End-User License & Agreement dengan saksama. Kemudian jika sesuai keinginan, pasang software tersebut.

6. Software sudah siap dipakai, jangan lupa untuk senantiasa mengecek sumber software seperti website untuk menerima update & perbaikan bug terbaru.

Semoga membantu.

15. Jelaskan cara menginstal software sistem operasi!


Kelas: X

Mata Pelajaran: Teknologi Informasi

Materi: Sistem Operasi

Kata Kunci: Instalasi Sistem Operasi

 

Pembahasan:

 

Cara menginstal software sistem operasi:

 

1. Cara lama dengan melakukan booting dari media penyimpanan Sistem operasi

 

Sistem operasi lama, seperti Windows XP, Windows Vista, Windows 7 dan Linux versi lama harus diinstal pada saat komputer melakukan proses booting. Proses booting adalah proses saat komputer dinyalakan dimana komputer akan mengunduh data perangkat lunak dan sistem operasi dari media penyimpanan.

 

Pada kondisi biasa, komputer akan mengunduh sistem operasi yang sudah terinstal dan tersimpan di hard disk. Namun kita bisa menginterupsi atau mengalihkan proses booting ini, dengan menekan Esc atau F1 (tergantung komputer), lalu memilih lokasi data Sistem operasi yang diinstall, sehingga komputer akan mengunduh media penyimpanan sistem operasi yang ingin diinstal. Misalnya mengunduh dari USB Flash Disk, dari CD atau dari DVD.

 

Setelah mengunduh data sistem operasi yang akan diinstal, akan muncul beberapa dialog tentang instalasi ini, seperti lokasi partisi hard disk yang akan digunakan dan pilihan instalasi.

 

Sistem operasi versi lama memerlukan hard disk yang akan digunakan untuk diformat bersih, sementara sistem operasi terbaru bisa diinstal tanpa haru melakukan format.

 

2. Menggunakan Media Creation Tool

 

Pada sistem operasi terbaru Windows, yaitu Windows 10, pengguna dapat melakukan instalasi dengan tanpa melakukan booting, bila komputer sudah terinstall versi Windows yang lama. Pengguna hanya perlu menjalankan porgram yang disebut  “Media Creation Tool”, kemudian memilih “Upgrade This PC”, lalu memilih opsi yang diinginkan (bahasa, edisi, jenis 32 atau 64 bitdan sebagainya), lalu menjalankan program hingga Windows 10 terinstal.
