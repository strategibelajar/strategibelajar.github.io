c. f(-2) = -4

f(x) = ax + b

f(3) = 3a + b = 1

f(1). = a + b = -1

a + b = -1

1 + b = -1

b = -1 - 1 = -2

c. f(-2) = -2 - 2 = -4

f(x) = ax + b

f(3) = 1 ➡ 3a + b = 1

f(1) = -1 ➡ a + b = -1

_______________–

2a = 2

a = 1

Substitusikan nilai ke dalam salah satu persamaan

a + b = -1

1 + b = -1

b = -1 - 1

b = -2

b. Bentuk fungsi ➡ f(x) = x - 2

c. Nilai dari f(-2)
