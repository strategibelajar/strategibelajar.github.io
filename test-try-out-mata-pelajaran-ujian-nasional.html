17. Empat orang anak yang bernama Rudi, Gilang, Yoga, dan Bima mengikuti Try Out Mata Pelajaran Matematika di sekolahnya. Jika rata-rata nilai Try Out Rudi dan Gilang adalah 50, rata-rata nilai Try Out Gilang dan Yoga adalah 75, serta rata-rata nilai Try Out Yoga dan Bima adalah 70. Berapakah rata-rata nilai Try Out Rudi dan Bima?​


Untuk menemukan rata-rata nilai Try Out Rudi dan Bima, kita perlu menggunakan informasi yang diberikan.

Kita diberi tiga persamaan:

(Rudi + Gilang) / 2 = 50

(Gilang + Yoga) / 2 = 75

(Yoga + Bima) / 2 = 70

Kita selesaikan persamaan ini untuk mencari nilai rata-rata Try Out Rudi dan Bima.

Dari persamaan 1

(Rudi + Gilang) / 2 = 50, kita bisa mendapatkan Rudi + Gilang = 100.

Dari persamaan 2

(Gilang + Yoga) / 2 = 75, kita bisa mendapatkan Gilang + Yoga = 150.

Dari persamaan 3

(Yoga + Bima) / 2 = 70, kita bisa mendapatkan Yoga + Bima = 140.

Kita ingin mencari rata-rata Rudi dan Bima, yaitu (Rudi + Bima) / 2.

Jika kita mengurangi persamaan Gilang + Yoga = 150 dari persamaan Rudi + Gilang = 100, maka kita mendapatkan (Rudi - Yoga) = -50.

Jika kita mengurangi persamaan Yoga + Bima = 140 dari persamaan Gilang + Yoga = 150, maka kita mendapatkan (Gilang - Bima) = 10.

Jadi, kita memiliki sistem persamaan:

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50

Jadi, kita memiliki sistem persamaan:Rudi - Yoga = -50Gilang - Bima = 10

Kita ingin mencari nilai (Rudi + Bima) / 2. Untuk mencapainya, kita dapat menambah persamaan di atas:

(Rudi - Yoga) + (Gilang - Bima) = (-50) + 10

Rudi + Bima = -40

Akhirnya, kita dapat mencari rata-rata nilai Try Out Rudi dan Bima:

(Rudi + Bima) / 2 = (-40) / 2 = -20

Jadi, rata-rata nilai Try Out Rudi dan Bima adalah -20.
