2. cos (2° - 3) cos (° + 2) = 0​


Jawaban:

Persamaan Trigonometri

cos 2x - 3 cos x + 2 = 0

⇔ (2 cos² x - 1) - 3 cos x + 2 = 0

⇔ 2 cos² x - 3 cos x + 1 = 0

⇔ (2 cos x - 1)(cos x - 1) = 0

⇔ cos x =1/2 atau cos x = 1

cos x = 1/2 → x = 60, 300

cos x = 1 → x = 0, 360

Jadi, Himpunan penyelesaian persamaan tersebut adalah {0°, 60°, 300°, 360°}

6. cos 2x -3 cos x +2 = 0


[tex]\cos{2x}-3\cos{x}+2=0[/tex]

Gunakan rumus sudut ganda untuk cos:
[tex]\cos{2x}=2\cos^2{x}-1 \implies \\ \\ 2\cos^2{x}-3\cos{x}+1 = 0[/tex]

Faktorkan polinomial tersebut:
[tex]\displaystyle (\cos{x}-1)\left(\cos{x}-\frac{1}{2}\right) = 0 \implies \\ \\ \cos{x} = 1, \quad \cos{x} = \frac{1}{2}[/tex]

Kita mendapatkan dua solusi untuk [tex]\cos{x}[/tex], untuk tiap jawaban - bisa didapat nilai dari [tex]x[/tex]:
[tex]\cos{x} = 1 \implies x = 0 + 2\pi k, \quad k \in \mathbb{Z} \\ \cos{x} = \frac{1}{2} \implies x = \frac{\pi}{3} + 2\pi k, \quad k \in \mathbb{Z}[/tex]

Jadi semua solusinya adalah:
[tex]x = 2\pi k \text{ atau } x = \frac{\pi}{3} + 2\pi k , \quad k \in \mathbb{Z}[/tex]

35. 2 cos² x - 3 cos x - 3 = 0​


Untuk menyelesaikan persamaan kuadrat seperti ini, kita dapat menggunakan substitusi dengan memperkenalkan variabel baru. Mari kita substitusi cos(x) dengan variabel baru, misalnya t = cos(x).

Maka persamaan menjadi:

2t² - 3t - 3 = 0

Kemudian kita mencari akar-akarnya dengan menggunakan rumus kuadrat:

t = [-(-3) ± √((-3)² - 4(2)(-3))] / (2 * 2)

t = [3 ± √(9 + 24)] / 4

t = [3 ± √33] / 4

Sehingga, kita mendapatkan dua nilai t:

1. t = (3 + √33) / 4

2. t = (3 - √33) / 4

Namun, ingat bahwa kita melakukan substitusi sebelumnya, yaitu t = cos(x). Jadi, akar-akar persamaan aslinya adalah:

1. cos(x) = (3 + √33) / 4

2. cos(x) = (3 - √33) / 4

Untuk mendapatkan nilai x, kita perlu mengambil arccos dari kedua sisi persamaan:

1. x = arccos((3 + √33) / 4)

2. x = arccos((3 - √33) / 4)

Sehingga, solusi dari persamaan 2 cos² x - 3 cos x - 3 = 0 adalah x = arccos((3 + √33) / 4) dan x = arccos((3 - √33) / 4).

Menyelesaikan persamaan 2 cos² x - 3 cos x - 3 = 0, kita dapat menggunakan rumus kuadrat. Mari kita tulis ulang persamaan tersebut dalam bentuk kuadrat standar:

2 cos² x - 3 cos x - 3 = 0 2 cos² x - 6 cos x + 3 cos x - 3 = 0 (2 cos x - 3)(cos x + 1) = 0

Jadi, baik (2 cos x - 3) = 0 atau (cos x + 1) = 0.

Jika (2 cos x - 3) = 0, maka:

2 cos x = 3 cos x = 3/2

Ini tidak mungkin karena fungsi kosinus hanya mengambil nilai antara -1 dan 1.

Jika (cos x + 1) = 0, maka:

cos x = -1

Ini terjadi ketika x adalah kelipatan ganjil dari π.

Oleh karena itu, solusi untuk persamaan tersebut adalah:

x = (2n + 1)π, di mana n adalah bilangan bulat.
