18. setarakan persamaan reaksi berikut dengan cara matematika! Fe(s) + H2SO4(aq) = Fe2(SO4)3(aq) + H2(g)


Persamaan reaksi setara berdasarkan pada cara matematikanya adalah:

2 Fe(s)  + 3 H₂SO₄(aq)  ==>  Fe₂(SO₄)₃(aq)  +  3 H₂(g)

Penjelasan:

Penyetaraan Reaksi kimia sederhana adalah langkah-langkah untuk menyamakan jumlah atom yang mengalami reaksi kimia sehingga jumlah atom pada reaktan yang bereaksi sama dengan jumlah atom pada produk yang dihasilkan.

Proses penyetaraan reaksi kimia dilakukan dengan menambahkan Koefisien reaksi yaitu angka yang ditambahkan di depan rumus molekul dari setiap spesi yang bereaksi.

Berdasarkan pada penjelasan diatas, maka kita coba terapkan untuk menyetarakan reaksi kimia berikut menggunakan cara matematika yaitu:

a Fe(s)  + b H₂SO₄(aq)  ==>  Fe₂(SO₄)₃(aq)  +  c H₂(g)

* Kita misalkan koefisien reaksi dari setiap senyawa dengan huruf a, b, dan c

** Untuk senyawa yang paling kompleks atau paling besar kita misalkan nilai koefisiennya adalah 1 dalam kasus ini adalah Fe₂(SO₄)₃ (Koefisien 1 tidak perlu dituliskan)

Selanjutnya berdasarkan persamaan reaksi diatas maka kita tuliskan persamaan matematikanya dimana reaksi dikatakan setara jika jumlah atom di kiri (Reaktan) adalah sama dengan di kanan (Produk) sehingga:

#Atom Fe

 Jumlah di kiri = a

 Jumlah di kanan = 2 x 1 = 2

 Sehingga kiri = kanan maka:

 a = 2

#Atom H

 Jumlah di kiri = 2 x b = 2b

 Jumlah di kanan = 2 x c = 2c

 Sehingga kiri = kanan maka:

 2b = 2c ==> b = c

#Atom S

 Jumlah di kiri = 1 x b = b

 Jumlah di kanan = 3 x 1 x 1 = 3

 Sehingga kiri = kanan maka:

 b = 3

#Atom O

  Jumlah di kiri = 4 x b = 4b

  Jumlah di kanan = 3 x 4 x 1 = 12

  Sehingga kiri = kanan  maka:

  4b = 12  ==>  b = 3

Karena untuk jumlah atom H diatas diketahui b = c dimana nilai b = 3 maka:

b = c = 3

sehingga setelah nilai a, b, dan c diketahui maka kita masukkan pada persamaan reaksi di atas yaitu:

a Fe(s)  + b H₂SO₄(aq)  ==>  Fe₂(SO₄)₃(aq)  +  c H₂(g)

menjadi

2 Fe(s)  + 3 H₂SO₄(aq)  ==>  Fe₂(SO₄)₃(aq)  +  3 H₂(g) (SETARA)
