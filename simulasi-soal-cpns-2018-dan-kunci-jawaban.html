7. bantu jawab, ini dari soal simulasi UNBK kemarin


[tex]Persamaan\: Kuadrat [/tex]

Persamaan Kuadrat 2x² + mx + 16 = 0 memiliki akar-akar p dan q, jika p = 2q, sedangkan p, q > 0
maka m = ... ?

dari persamaan kuadrat tersebut,
2x² + mx + 16 = 0
a = 2
b = m
c = 16

p × q = c/a = 8
(2q) × q = 8
2q² = 8
q = ±4, karena q > 0, maka q = 4

maka p = 2(4) = 8
sehingga
p + q = -b/a
8 + 4 = -m/2
12 = -m/2
m = -24 ✔️

keyword : akar persamaan kuadrat, jumlah akar, hasil kali akar
