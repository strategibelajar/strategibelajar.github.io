NaOH 8% dgn massa jenis 1.25 g/mL
Larutan NaOH 0.05 M dgn vol. = 2 L

Ditanya:
Vol. NaOH 8% yg dibutuhkan dan jumlah air yg ditambahkan ???

Pembahasan:

Mr NaOH = 40

1. Hitung mol NaOH dlm 2L 0.05 M

NaOH: 2 L × 0.05 mol/L = 0.1 mol

2. Jumlah NaOH ini (pd no. 1) berasal dari x mL NaOH 8% (%berat/vol) dan penambahan y mL air, sehingga

(x mL × 1.25 g/mL × 8%)/40 g/mol = 0.1 mol

didapatkan x = 40 mL dlm 2000 mL air

Jadi air yg ditambahkan:
2000 - 40 = 1960 mL

agar f(x) naik maka f'(x)>0

f(x) = √(x⁴+4x²)
f(x) = (x⁴+4x²)¹/²
f'(x) = 1/2 . (x⁴+4x²)¹/²⁻¹ > 0
f'(x) = 1/2 . (x⁴+4x²)⁻¹/² > 0
f'(x) = 1/(2√(x⁴+4x²)) > 0

2√(x⁴+4x²) > 0
√(x⁴+4x²) > 0
(x⁴+4x²) > 0
x²(x²+4) >0
x²>0
x>0 atau x<0

jadi f(x) naik pada interval x>0 dan x<0

anggap aja

n = bnyak tryout

x = jmlah nilai tryout sebelumnya

berarti :

Xn + 80 / (n+1) = 85..

Xn + 80 = 85n + 85

Xn = 85n + 5...(1)

Xn + 95 /(n+1) = 88

Xn + 95 = 88n + 88

Xn = 88n - 7...(2)

eliminasi deh.

Xn = 85n + 5

Xn = 88n - 7

---------------------- -

0 = -3n + 12

-3n = -12

n = 12/3

n = 4
