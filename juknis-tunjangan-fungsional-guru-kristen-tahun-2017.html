Tunjangan istri = 10% x gaji pokok = 10% x Rp 6.500.000 = Rp 650.000

2. Tunjangan anak:

Tunjangan anak = 2% x jumlah anak x gaji pokok = 2% x 3 x Rp 6.500.000 = Rp 390.000

Berdasarkan data di atas, anda diminta: Menghitung PPh terutang atas gaji Drs. Hanung Wicaksana untuk tahun 2017.​


Jawaban:

Penjelasan:

Untuk menghitung PPh terutang atas gaji Drs. Hanung Wicaksana untuk tahun 2017, kita perlu mengikuti langkah-langkah berikut:

1. Menghitung Pendapatan Bruto (PB):

  Pendapatan Bruto (PB) = Gaji Pokok + Tunjangan Istri + Tunjangan Fungsional

  PB = Rp 4.000.000 + (10% x Rp 4.000.000) + Rp 3.000.000

  PB = Rp 4.000.000 + Rp 400.000 + Rp 3.000.000

  PB = Rp 7.400.000

2. Menghitung Iuran Pensiun (IP):

  Iuran Pensiun (IP) = 4,75% x Pendapatan Bruto

  IP = 4,75% x Rp 7.400.000

  IP = Rp 351.500

3. Menghitung Penghasilan Netto (PN):

  Penghasilan Netto (PN) = Pendapatan Bruto - Iuran Pensiun

  PN = Rp 7.400.000 - Rp 351.500

  PN = Rp 7.048.500

4. Menghitung Penghasilan Netto Setahun (PNS):

  Penghasilan Netto Setahun (PNS) = Penghasilan Netto x 12 bulan

  PNS = Rp 7.048.500 x 12

  PNS = Rp 84.582.000

5. Menghitung Penghasilan Kena Pajak (PKP):

  Penghasilan Kena Pajak (PKP) = Penghasilan Netto Setahun - PTKP

  PTKP = Rp 39.600.000 (untuk tahun 2017)

  PKP = Rp 84.582.000 - Rp 39.600.000

  PKP = Rp 44.982.000

6. Menghitung PPh Terutang:

  PPh Terutang = (PKP x Tarif Pajak) - Pengurang Pajak

  Tarif Pajak (berdasarkan tabel PPh 21) = 5%

  Pengurang Pajak = Rp 500.000 (untuk tahun 2017)

  PPh Terutang = (Rp 44.982.000 x 5%) - Rp 500.000

  PPh Terutang = Rp 2.249.100 - Rp 500.000

  PPh Terutang = Rp 1.749.100

Jadi, PPh terutang atas gaji Drs. Hanung Wicaksana untuk tahun 2017 adalah sebesar Rp 1.749.100.
