3. Si Amir mendapat pekerjaan baru, dan akan memperoleh gaji per bulan dengan memilih satu dari dua cara penggajian berikut ini. Cara penggajian pertama: Gaji pokok Rp 600.000,00 ditambah komisi 4% dari hasil penjualan. Cara penggajian kedua: Gaji pokok Rp 800.000,00 ditambah komisi 6% dari hasil penjualan setelah dikurangi Rp 10.000.000,00. Selidikilah cara penggajian manakah yang lebih menguntungkan si Amir? (butuh memakai cara)


Misalkan besar hasil penjualan adalah x rupiah :
sistem gaji pertama :
600.000 + 4% x = 600.000 + 0,04 x

sistem gaji kedua :
800.000 + 6% ( x - 10.000.000 )
= 800.000 + 0,06 ( x - 10.000.000 )
= 800.000 + 0,06 x - 600.000
= 200.000 + 0,06 x

600.000 + 0,04 x > 200.000 + 0,06 x
-0,02 x > -400.000
0,02 x < 400.000
x > 20.000.000

berdasarkan uraian diatas , dapat disimpulkan bahwa :
- sistem gaji pertama akan lebih menguntungkan jika besranya hasil penjualan
kurang dari Rp20.000.000,00
- sistem gaji kedua akan lebih menguntungkan jika besarnya hasil penjualan lebih dari Rp20.000.000,00
- sistem gaji pertama dan kedua akan memberikan gaji yang sama besar jika besarnya hasil penjualan adalah Rp20.000.000,00

18. gaji upah tunjangan honorarium komisi dan bonus termasuk ke dalam jenis penghasilan apa


Gaji, upah, tunjangan, honorarium, komisi, serta bonus termasuk jenis penghasilan yang menjadi objek pajak.  Hal ini tercantum pada UU Harmonisasi Pajak Nomor 7 Tahun 2021, Bab III Pasal 4.

Pembahasan

Dijelaskan dalam UU HPP  Bab III Pasal 4 Ayat 1 bahwa objek pajak adalah penghasilan yaitu setiap tambahan kemampuan ekonomis yang diterima atau diperoleh wajib pajak. Tambahan ini dapat dipakai untuk konsumsi atau menambah kekayaan wajib pajak bersangkutan, dengan nama apapun.

Secara detail yang menjadi objek pajak adalah:

Imbalan berkenaan dengan pekerjaan atau jasa yang diterima: gaji, upah, tunjangan, honorarium, komisi, bonus, gratifikasi, uang pensiuan dan lainnya termasuk natura. Hadiah undian, hadiah pekerjaan, hadiah suatu kegiatan dan penghargaan.Laba usaha.Keuntungan karena penjualan atau pengalihan harta. Penerimaan kembali pembayaran pajak yang dibebankan sebagai biaya dan pembayaran tambahan pengembalian pajak. Bunga termasuk premi, diskonto dan imbalan karena jaminan pengembalian utang. Dividen dan royalti.Sewa dan penghasilan lain yang berkaitan dengan penggunaan harta.Penerimaan atau pembayaran berkala.Keuntungan pembebasan utang dan keuntungan selisih kurs. Premi asuransi. Iuran yang diterima dalam menjalankan usaha.Tambahan kekayaan neto yang berasal dari penghasilan belum dikenai pajak. Penghasilan syariah. Imbalan bunga sesuai aturan perpajakan dan surplus Bank Indonesia.

23. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

33. si amir mendapat pekerjaan baru dan akan memperoleh gaji per bulan dengan memilih salah satu dari dua cara penggajian berikut ini. cara penggajian pertama : gaji pokok rp 600.000,00 ditambah komisi 4% dari hasil penjualan. cara penggajian kedua : gaji pokok rp 800.000,00 ditambah komisi 6% dari hasil penjualan setelah dikurangi rp 10.000.000,00 selidikilah cara penggajian manakah yang lebih mengguntungkan si amir


Misalkan besar hasil penjualan adalah x rupiah.

Sistem gaji pertama 
→ 600.000 + 4% x = 600.000 + 0,04 x

Sistem gaji kedua
→ 800.000 + 6% (x - 10.000.000)
     = 800.000 + 0,06 (x - 10.000.000)
     = 800.000 + 0,06 x - 600.000
     = 200.000 + 0,06 x

600.000 + 0,04 x > 200.000 + 0,06 x
-0,02 x > -400.000
0,02 x < 400.000
x < 20.000.000 
600.000 + 0,04 x < 200.000 + 0,06 x
-0,02 x < -400.000
0,02 x > 400.000
x > 20.000.000

Berdasarkan uraian di atas, dapat disimpulkan bahwa:
- sistem gaji pertama akan lebih menguntungkan jika besarnya hasil penjualan kurang dari Rp20.000.000,00
- sistem gaji kedua akan lebih menguntungkan jika besarnya hasil penjualan lebih dari Rp20.000.000,00
- sistem gaji pertama dan kedua akan memberikan gaji yang sama besar jika besarnya hasil penjualan adalah Rp20.000.000,00
