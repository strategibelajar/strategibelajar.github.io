26. Tony telah mengikuti beberapa Try Out matematika yang diadakan oleh gurunya disekolah. Diketahui bahwa masih ada satu try out lagi yang harus diikuti Tony. Jika Tony mendapatkan nilai 80 pada try out berikutnya, rata-rata try out Tony adalah 85. Sedangkan jika Tony mendapatkan nilai 95 pada try out berikutnya, rata-ratanya menjadi 88. Berapa banyak jumlah Try Out seluruhnya yang diikuti Tony ?


anggap aja

n = bnyak tryout

x = jmlah nilai tryout sebelumnya

berarti :

Xn + 80 / (n+1) = 85..

Xn + 80 = 85n + 85

Xn = 85n + 5...(1)

Xn + 95 /(n+1) = 88

Xn + 95 = 88n + 88

Xn = 88n - 7...(2)

eliminasi deh.

Xn = 85n + 5

Xn = 88n - 7

---------------------- -

0 = -3n + 12

-3n = -12

n = 12/3

n = 4

berarti tony 4 x ngikut tryout..
