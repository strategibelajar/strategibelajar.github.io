8x = 240.000

Berapakah nilai tes matematika Bayu?


JAWABAN:

banyak soal = 40

nilai untuk soal yang benar = 4

nilai untuk soal yang salah = -1

nilai untuk soal tidak dijawab = 0

bayu menjawab : benar = 34 soal

salah = 2 soal

DITANYA:

Berapakah nilai tes matematika Bayu?

DIJAWAB:

nilai yang benar = 34 x 4

= 136

nilai yang salah = 2 x (-1)

= -2

nilai yang tidak dijawab = (40 - (34 + 2)) x 0

= (40 - 36) x 0

= 4 x 0

= 0

total nilai = 136 + (-2) + 0

= 134

jadi, nilai tes matematika Bayu adalah 134.

Situasi
