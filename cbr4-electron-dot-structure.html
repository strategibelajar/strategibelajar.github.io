27. Setarakan persamaan reaksi berikut. : CH4+Br2 CBr4+HBr


Jawaban:

CH4 + 4 Br2 => CBr4 + 4 HBr

Penjelasan:

Tambahkan abjad di tiap koefisien:

a CH4 + b Br2 => c CBr4 + d HBr

Tulis persamaan tiap atom:

C : a = c

H : 4a = d

Br : 2b = 4c + d

Misalkan a = 1, c = 1

4a = d, maka d = 4

2b = 4c + d

2b = 4.1 + 4 maka b = 4

Koefisien reaksi: a = 1 b = 4 c = 1 d = 4

Masukan ke persamaan reaksi:

CH4 + 4 Br2 => CBr4 + 4 HBr

kuta cari dulu (gofoh)nya, misal jawabanya (gofoh)(x) = x + 1
(gofoh) (x) sama saja f(x) dan sama saja dg y, jadi y = x+1.
kita pindah menjadi persamaan x=..
jadi, x = -y+1
tinggal kita tentukan invers nya, dg cara mengubah y menjadi x.
jadi, f^-1(x) = -x + 1
