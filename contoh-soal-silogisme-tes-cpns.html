15. Peluang seseorang dapat lolos tes cpns adalah 0,05. apabila terdapat 750.000 peserta tes cpns, jumlah peserta akan diterima adalah​


Jumlah peserta yang akan diterima adalah 37.500 peserta.

Penjelasan dengan langkah-langkah:

Frekuensi harapan adalah banyaknya kejadian yang diharapkan terjadi dari suatu peristiwa. Rumus untuk frekuensi harapan, yaitu:

[tex]Fh(A)=n. P(A)[/tex]

Dengan:

Fh(A) adalah frekuensi harapan dari kejadian A.n adalah banyaknya percobaan.P(A) adalah peluang dari kejadian A.

Diketahui:

P(A)= 0,05.n = 750.000

Ditanya:

Jumlah peserta yang akan diterima?

Jawab:

[tex]Fh(A)=n. P(A)[/tex]

[tex]Fh(A)=(750.000). (0,05) = 37.500[/tex]

Maka, banyaknya peserta yang akan diterima adalah 37.500 peserta.

18. Contoh soal penarikan kesimpulan yg sah dalam modus silogisme


Contoh soal : modus silogisme 1. Tentukan kesimpulan dari premis berikut ini :) Jika x bilangan real, maka x2 ≥ 0 Jika x2 ≥ 0, maka (x2 + 2) > 0 Pembahasan : Untuk mempermudah, lakukan pemisalan sebagai berikut : » x bilangan real = A » x2 ≥ 0 = B » (x2 + 2) > 0 = C Dengan menggunakan kaidah silogisme, maka pernyataan di atas dapat disusun menjadi : A ⇒ B B ⇒ C ∴ A ⇒ C Dengan demikian, kesimpulan dari premis di atas adalah : Jika x bilangan real, maka (x2 + 2) > 0. 2. Diketahui pernyataan sebagai berikut : Jika Rihanna konser di Jakarta, maka saya akan menonton Jika saya menonton, maka saya sangat senang Tentukan kesimpulan yang sah tentang pernyataan di atas. Pembahasan : Rihanna konser di Jakarta : P Saya menonton : Q Saya sangat senang : R Dengan menggunakan silogisme, maka pernyataan di atas dapat disusun menjadi : P ⇒ Q Q ⇒ R ∴ P ⇒ R Jadi, kesimpulan dari pernyataan di atas adalah : Jika Rihanna konser di Jakarta, maka saya sangat senang.

19. Contoh soal silogisme dalam matematila


Silogisme berarti “jika diketahu p → q dan q→r, maka bisa ditarik kesimpulan p→r“. sebagai contoh :
Premis 1 : Jika harga BBM naik, maka harga bahan pokok naik.
Premis 2 : Jika harga bahan pokok naik maka semua orang tidak senang
maka kesimpulannya adalah Jika harga BBM naik, maka semua orang tidak senang.

25. contoh silogisme negatif, silogisme yang salah, silogisme entimen, dan silogisme analogi?


Silogisme negatif:
PU= semua pejabat (tidak boleh) korupsi
PK= anjar adalah pejabat
K: Anjar (tidak boleh) korupsi

Silogisme yang salah
PU: semua siswa SD tidak mempunyai KTP
PK: Ari bukan siswa SD
K: Ari mempunyai KTP

Silogisme Entimen
Rumus: C=B, karena C=A
PU: semua anggota DPR sedang melakukan rapt
PK: Yanuar adalah anggota DPR
K: Yanuar sedang melakukan rapt
